## 1. 購入者向け画面遷移

### 1.1 全体構成

```
[未ログイン]
  C-01 トップページ
    ├── C-02 商品一覧ページ（検索結果兼用）
    │     └── C-03 商品詳細ページ
    │           ├── カートに追加（通常注文 or サンプル制作）（要ログイン）
    │           └── C-14 問い合わせフォーム（商品情報自動セット）
    ├── C-04 カートページ（要ログイン）
    │     └── C-05 入稿ページ（商品ごと）
    │           └── 全商品入稿完了 → C-06 注文確認ページ → C-07 注文完了ページ
    ├── C-08 ログイン / C-09 会員登録ページ
    ├── C-14 問い合わせフォーム（フッターからは空フォーム）
    ├── C-15 FAQページ
    ├── C-16 ご利用ガイド
    ├── C-17 利用規約
    ├── C-18 プライバシーポリシー
    └── C-19 特商法表記

[ログイン済み]
  C-10 マイページ
    ├── C-11 会員情報編集ページ
    └── C-12 注文履歴一覧ページ
          └── C-13 注文詳細ページ（ステータス確認・再入稿・リピート注文・共有URLコピー）

[メール経由]
  決済案内メール → 支払い方法に応じた決済（Stripe決済画面 / 銀行振込 / 請求書払い）
  ステータス変更通知メール → C-21 共有ステータスページ（ログイン不要）

[URL共有]
  C-21 共有ステータスページ（トークン付きURL。EC注文・代理注文共通）

[エラー]
  C-20 404エラーページ
```

| C-12 | 注文履歴一覧ページ | 必要 | 過去の注文一覧（ステータス表示）。通常注文・サンプル制作の両方を含む |

| **情報ページ** | | | |
| C-15 | FAQページ | 不要 | よくある質問と回答 |

| C-18 | プライバシーポリシー | 不要 | プライバシーポリシーの全文 |

### 1.4 画面遷移詳細

#### 商品閲覧〜カート追加フロー

```
C-01 トップページ
  │
  ├──[カテゴリ選択 / 検索]──→ C-02 商品一覧ページ
  │                              │
  │                              └──[商品選択]──→ C-03 商品詳細ページ
  │                                                 │
  │                                                 ├──[見積書DL]──→ PDF ダウンロード
  │                                                 ├──[テンプレートDL]──→ 入稿時テンプレートファイルダウンロード
  │                                                 ├──[問い合わせ]──→ C-14 問い合わせフォーム（商品情報自動セット）──→ 送信後 C-03へ
  │                                                 │
  │                                                 ├──[カートに追加]（通常注文）
  │                                                 │     └──→ カートに追加完了（モーダル表示）──→ C-03に留まる（※ログイン不要）
  │                                                 │
  │                                                 └──[サンプル制作をカートに追加]
  │                                                       └──→ サンプルとしてカートに追加（モーダル表示）──→ C-03に留まる（※ログイン不要）
  │
  └──[ログイン]──→ C-08 ログインページ
```

#### カート〜注文フロー

```
C-04 カートページ（ヘッダーのカートアイコンからアクセス）
  │
  │  カート内商品一覧:
  │  ┌──────────────────────────────────────────────────────┐
  │  │ [通常] 商品A  44mm / マット / 100個  8,800円  │ 未入稿 [入稿する]  │
  │  │ [サンプル] 商品B  56mm / グロス / 10個  750円  │ 入稿済 [変更]     │
  │  └──────────────────────────────────────────────────────┘
  │
  ├──[入稿する]
  │     │
  │     ├──(未ログイン)──→ C-08 ログイン ──→ C-05 入稿ページへ
  │     │
  │     └──(ログイン済み)──→ C-05 入稿ページ（該当商品の入稿）
  │                              │
  │                              └──[入稿完了]──→ C-04 カートページへ戻る
  │
  ├──[数量変更 / 商品削除]──→ カート更新（C-04内で即時反映）
  │
  └──[注文に進む]（※全商品が入稿済みの場合のみ有効）
        │
        └──→ C-06 注文確認ページ
                │
                └──[注文確定]──→ C-07 注文完了ページ ──→ C-02へ
```

#### 決済フロー（メール経由）

```
決済案内メール ──→ 支払い方法に応じた決済（クレカ: Stripe決済画面 / 振込: 振込先に入金 / 請求書: 月末締め翌月払い）
```

> **備考:** 決済はStripe Payment Links / Checkoutを利用。自サイト内に決済画面は持たない。

  ├──[会員情報]──→ C-11 会員情報編集ページ
  │
  └──[注文履歴]──→ C-12 注文履歴一覧ページ
                      │
                      └──[注文選択]──→ C-13 注文詳細ページ
                                          │
                                          ├──[再入稿]──→ C-05 入稿ページ（再入稿モード）
                                          │
                                          └──[リピート注文]──→ カートに追加 ──→ C-04 カートページ
```

#### 会員登録フロー

```
C-08 ログインページ
  │
  ├──[ソーシャルログイン（Google / Apple / Facebook / X）]──→ 各プロバイダー認証
  │                         │
  │                         ├──(初回)──→ C-09 会員登録ページ（お客様情報入力 or スキップ）──→ C-10 マイページ

                                                                          │

## 2. 管理者向け画面遷移

### 2.1 全体構成

```
A-01 ログイン ──→ A-02 ダッシュボード
                      ├── 商品管理
                      │     ├── A-03 商品一覧
                      │     ├── A-04 商品登録
                      │     └── A-05 商品編集
                      ├── 注文管理
                      │     ├── A-06 注文一覧（通常注文・サンプル制作・代理注文を含む。チャネル別フィルタ）
                      │     ├── A-07 注文詳細（入稿確認・承認・差し戻し・決済案内送信・ステータス更新・共有URL発行）
                      │     └── 代理注文作成（EC外注文の手動登録。A-07内 or 専用フォーム）
                      ├── 会員管理
                      │     ├── A-08 会員一覧
                      │     └── A-09 会員詳細
                      ├── コンテンツ管理
                      │     ├── A-10 FAQ管理（一覧上でCRUD）
                      │     ├── ヒーローバナー管理（C-01カルーセルの画像・リンク設定）
                      │     ├── おすすめ商品管理（おすすめ度設定）
                      │     └── カテゴリ管理（追加・編集・並び順・画像）
                      └── 設定
```

### 2.2 画面一覧

| ID | 画面名 | 概要 |
|----|--------|------|
| A-01 | ログインページ | 管理者認証 |
| A-02 | ダッシュボード | 新規注文・入稿通知の一覧、主要メニューへの導線 |
| A-03 | 商品一覧 | 登録済み商品の一覧。公開/非公開切替 |
| A-04 | 商品登録 | 新規商品の登録（画像、説明、オプション、価格テーブル、納期、テンプレート等） |
| A-05 | 商品編集 | 既存商品の編集 |
| A-06 | 注文一覧 | 全注文のステータス別一覧。通常注文・サンプル制作・代理注文を含む。チャネル別（EC/電話/メール/営業）フィルタ。CSV出力 |
| A-07 | 注文詳細 | 注文内容、注文連絡先（メール・電話）、入稿データプレビュー、承認/差し戻し、決済案内送信、ステータス更新、共有ステータスURL表示・コピー、仕上がりイメージアップロード、担当者割り当て |

| A-10 | FAQ管理 | FAQ一覧上で追加・編集・削除をインライン操作 |
