# 画面遷移図：推し活グッズ BtoB ECサイト

> 最終更新日: 2026-04-13
> バージョン: 2.1.0

---

## 1. 購入者向け画面遷移

### 1.1 全体構成

```
[未ログイン]
  C-01 トップページ
    ├── C-02 商品一覧ページ（検索結果兼用）
    │     └── C-03 商品詳細ページ
    │           ├── カートに追加（通常注文 or サンプル制作）（要ログイン）
    │           └── C-14 商品問い合わせフォーム
    ├── C-04 カートページ（要ログイン）
    │     └── C-05 入稿ページ（商品ごと）
    │           └── 全商品入稿完了 → C-06 注文確認ページ → C-07 注文完了ページ
    ├── C-08 ログイン / C-09 会員登録ページ
    ├── C-15 一般問い合わせフォーム
    ├── C-16 FAQページ
    ├── C-17 ご利用ガイド
    ├── C-18 利用規約
    ├── C-19 プライバシーポリシー
    └── C-20 特商法表記

[ログイン済み]
  C-10 マイページ
    ├── C-11 会員情報編集ページ
    └── C-12 注文履歴一覧ページ
          └── C-13 注文詳細ページ（ステータス確認・再入稿・リピート注文）

[メール経由]
  決済リンク → Stripe決済画面（外部）→ Stripe完了画面

[エラー]
  C-21 404エラーページ
```

### 1.2 画面一覧

| ID | 画面名 | 認証 | 概要 |
|----|--------|------|------|
| **商品閲覧** | | | |
| C-01 | トップページ | 不要 | サービス紹介、カテゴリ一覧、おすすめ商品等の導線 |
| C-02 | 商品一覧ページ | 不要 | カテゴリ別の商品一覧。検索結果も兼ねる。検索・絞り込み・並び替え |
| C-03 | 商品詳細ページ | 不要 | 商品情報、画像（オプション連動切替）、オプション選択、数量入力、即時見積もり、納期目安、価格テーブル、テンプレートDL、見積書DL |
| **カート・注文** | | | |
| C-04 | カートページ | 必要 | カート内商品一覧（通常注文・サンプル制作を種別表示）、各商品の入稿状態表示、入稿ページへの導線、合計金額、注文確認への導線（全商品入稿完了時のみ有効） |
| C-05 | 入稿ページ | 必要 | 商品ごとのデザインデータアップロード（複数ファイル）、ライセンス同意チェック |
| C-06 | 注文確認ページ | 必要 | カート内全商品の注文内容（商品・オプション・数量・入稿データ・価格）の最終確認、注文確定ボタン |
| C-07 | 注文完了ページ | 必要 | 注文受付完了メッセージ、注文番号表示。商品一覧ページへの導線 |
| **認証** | | | |
| C-08 | ログインページ | 不要 | Supabase Auth。ソーシャルログイン（Google、Apple、Facebook、X）＋マジックリンク（メール） |
| C-09 | 会員登録ページ | 不要 | 初回ログイン時のみ表示。会社名・担当者名・電話番号・住所を入力 |
| **マイページ** | | | |
| C-10 | マイページ | 必要 | 会員情報、注文履歴へのハブ |
| C-11 | 会員情報編集ページ | 必要 | 会社名、担当者名、電話番号、住所等の編集 |
| C-12 | 注文履歴一覧ページ | 必要 | 過去の注文一覧（ステータス表示）。通常注文・サンプル制作の両方を含む |
| C-13 | 注文詳細ページ | 必要 | 注文内容、入稿データ、ステータス、再入稿・リピート注文ボタン |
| **問い合わせ** | | | |
| C-14 | 商品問い合わせフォーム | 不要 | 商品ページからの問い合わせ。商品情報が自動セット。スパム対策（reCAPTCHA等）あり |
| C-15 | 一般問い合わせフォーム | 不要 | 商品に紐づかない一般的な問い合わせ。スパム対策（reCAPTCHA等）あり |
| **情報ページ** | | | |
| C-16 | FAQページ | 不要 | よくある質問と回答 |
| C-17 | ご利用ガイド | 不要 | 注文〜発送の流れを説明 |
| C-18 | 利用規約 | 不要 | 利用規約の全文 |
| C-19 | プライバシーポリシー | 不要 | プライバシーポリシーの全文 |
| C-20 | 特商法表記 | 不要 | 特定商取引法に基づく表記 |
| **エラー** | | | |
| C-21 | 404エラーページ | 不要 | 存在しないページへのアクセス時に表示。トップページへの導線 |

### 1.3 URL構成

```
/                           → C-01 トップページ
/products                   → C-02 商品一覧（全商品）
/products?category=xxx      → C-02 カテゴリ絞り込み
/products?q=xxx             → C-02 検索結果
/products/[slug]            → C-03 商品詳細ページ
/cart                       → C-04 カートページ
/cart/upload/[cart-item-id]  → C-05 入稿ページ（カート内商品ごと）
/cart/confirm               → C-06 注文確認ページ
/cart/complete              → C-07 注文完了ページ
/login                      → C-08 ログインページ
/register                   → C-09 会員登録ページ
/mypage                     → C-10 マイページ
/mypage/profile             → C-11 会員情報編集ページ
/mypage/orders              → C-12 注文履歴一覧ページ
/mypage/orders/[id]         → C-13 注文詳細ページ
/contact?product=[slug]     → C-14 商品問い合わせフォーム
/contact                    → C-15 一般問い合わせフォーム
/faq                        → C-16 FAQページ
/guide                      → C-17 ご利用ガイド
/terms                      → C-18 利用規約
/privacy                    → C-19 プライバシーポリシー
/legal                      → C-20 特商法表記
```

### 1.4 画面遷移詳細

#### 商品閲覧〜カート追加フロー

```
C-01 トップページ
  │
  ├──[カテゴリ選択 / 検索]──→ C-02 商品一覧ページ
  │                              │
  │                              └──[商品選択]──→ C-03 商品詳細ページ
  │                                                 │
  │                                                 ├──[見積書DL]──→ PDF ダウンロード
  │                                                 ├──[テンプレートDL]──→ 入稿時テンプレートファイルダウンロード
  │                                                 ├──[問い合わせ]──→ C-14 商品問い合わせフォーム ──→ 送信後 C-03へ
  │                                                 │
  │                                                 ├──[カートに追加]（通常注文）
  │                                                 │     │
  │                                                 │     ├──(未ログイン)──→ C-08 ログイン ──→ C-03へ（カートに追加）
  │                                                 │     │
  │                                                 │     └──(ログイン済み)──→ カートに追加完了（モーダル表示）──→ C-03に留まる
  │                                                 │
  │                                                 └──[サンプル制作をカートに追加]
  │                                                       │
  │                                                       ├──(未ログイン)──→ C-08 ログイン ──→ C-03へ（カートに追加）
  │                                                       │
  │                                                       └──(ログイン済み)──→ サンプルとしてカートに追加（モーダル表示）──→ C-03に留まる
  │
  └──[ログイン]──→ C-08 ログインページ
```

#### カート〜注文フロー

```
C-04 カートページ（ヘッダーのカートアイコンからアクセス）
  │
  │  カート内商品一覧:
  │  ┌──────────────────────────────────────────────────────┐
  │  │ [通常] 商品A  44mm / マット / 100個  8,800円  │ 未入稿 [入稿する]  │
  │  │ [サンプル] 商品B  56mm / グロス / 10個  750円  │ 入稿済 [変更]     │
  │  └──────────────────────────────────────────────────────┘
  │
  ├──[入稿する]──→ C-05 入稿ページ（該当商品の入稿）
  │                    │
  │                    └──[入稿完了]──→ C-04 カートページへ戻る
  │
  ├──[数量変更 / 商品削除]──→ カート更新（C-04内で即時反映）
  │
  └──[注文に進む]（※全商品が入稿済みの場合のみ有効）
        │
        └──→ C-06 注文確認ページ
                │
                └──[注文確定]──→ C-07 注文完了ページ ──→ C-02へ
```

#### 決済フロー（メール経由）

```
メール内の決済リンク ──→ Stripe決済画面（外部）──→ Stripe完了画面
```

> **備考:** 決済はStripe Payment Links / Checkoutを利用。自サイト内に決済画面は持たない。

#### マイページフロー

```
C-10 マイページ
  │
  ├──[会員情報]──→ C-11 会員情報編集ページ
  │
  └──[注文履歴]──→ C-12 注文履歴一覧ページ
                      │
                      └──[注文選択]──→ C-13 注文詳細ページ
                                          │
                                          ├──[再入稿]──→ C-05 入稿ページ（再入稿モード）
                                          │
                                          └──[リピート注文]──→ カートに追加 ──→ C-04 カートページ
```

#### 会員登録フロー

```
C-08 ログインページ
  │
  ├──[ソーシャルログイン（Google / Apple / Facebook / X）]──→ 各プロバイダー認証
  │                         │
  │                         ├──(初回)──→ C-09 会員登録ページ（会社情報入力）──→ C-10 マイページ
  │                         └──(2回目以降)──→ C-10 マイページ
  │
  └──[マジックリンク]──→ メールアドレス入力 ──→ ログインリンク送信 ──→ メール内リンク
                                                                          │
                                                                          ├──(初回)──→ C-09 会員登録ページ（会社情報入力）──→ C-10 マイページ
                                                                          └──(2回目以降)──→ C-10 マイページ
```

### 1.5 共通ナビゲーション

全ページに共通で表示されるナビゲーション要素：

**ヘッダー:**
- ロゴ（トップページへ）
- 商品カテゴリメニュー
- 検索バー
- カートアイコン（商品数バッジ表示）
- ログイン / マイページボタン

**フッター:**
- FAQ
- ご利用ガイド
- 利用規約
- プライバシーポリシー
- 特商法表記
- 問い合わせ（C-15 一般問い合わせ）
- コーポレートサイトリンク

## 2. 管理者向け画面遷移

### 2.1 全体構成

```
A-01 ログイン ──→ A-02 ダッシュボード
                      ├── 商品管理
                      │     ├── A-03 商品一覧
                      │     ├── A-04 商品登録
                      │     └── A-05 商品編集
                      ├── 注文管理
                      │     ├── A-06 注文一覧（通常注文・サンプル制作を含む）
                      │     └── A-07 注文詳細（入稿確認・承認・差し戻し・決済リンク送信・ステータス更新・承諾書DL）
                      ├── 会員管理
                      │     ├── A-08 会員一覧
                      │     └── A-09 会員詳細
                      └── FAQ管理
                            └── A-10 FAQ管理（一覧上でCRUD）
```

### 2.2 画面一覧

| ID | 画面名 | 概要 |
|----|--------|------|
| A-01 | ログインページ | 管理者認証 |
| A-02 | ダッシュボード | 新規注文・入稿通知の一覧、主要メニューへの導線 |
| A-03 | 商品一覧 | 登録済み商品の一覧。公開/非公開切替 |
| A-04 | 商品登録 | 新規商品の登録（画像、説明、オプション、価格テーブル、納期、テンプレート等） |
| A-05 | 商品編集 | 既存商品の編集 |
| A-06 | 注文一覧 | 全注文のステータス別一覧。通常注文・サンプル制作をフィルター切替。CSV出力 |
| A-07 | 注文詳細 | 注文内容、入稿データプレビュー、承認/差し戻し、決済リンク送信、ステータス更新、ライセンス承諾書PDF発行 |
| A-08 | 会員一覧 | 登録会員の一覧・検索 |
| A-09 | 会員詳細 | 会員情報の閲覧、注文履歴の参照 |
| A-10 | FAQ管理 | FAQ一覧上で追加・編集・削除をインライン操作 |

### 2.3 画面遷移詳細

#### 注文管理フロー（メイン業務）

```
A-02 ダッシュボード
  │
  └──[新規注文通知 / 注文管理]──→ A-06 注文一覧
                                      │
                                      └──[注文選択]──→ A-07 注文詳細
                                                          │
                                                          ├──[入稿データプレビュー]──→ データ表示
                                                          ├──[承認]──→ ステータス更新（承認済み）
                                                          ├──[差し戻し]──→ 差し戻しメール送信（NTF-004）
                                                          ├──[決済リンク送信]──→ Stripe決済リンク生成・メール送信（NTF-005）
                                                          ├──[ステータス更新]──→ 製造中 / 発送済み / 完了
                                                          └──[承諾書発行]──→ ライセンス承諾書PDFダウンロード
```

#### 商品管理フロー

```
A-02 ダッシュボード
  │
  └──[商品管理]──→ A-03 商品一覧
                      │
                      ├──[新規登録]──→ A-04 商品登録 ──→ A-03へ
                      ├──[編集]──→ A-05 商品編集 ──→ A-03へ
                      └──[公開/非公開切替]──→ 即時反映
```

#### 会員管理フロー

```
A-02 ダッシュボード
  │
  └──[会員管理]──→ A-08 会員一覧
                      │
                      └──[会員選択]──→ A-09 会員詳細（会員情報・注文履歴の参照）
```

#### FAQ管理フロー

```
A-02 ダッシュボード
  │
  └──[FAQ管理]──→ A-10 FAQ管理
                      │
                      ├──[新規作成]──→ 一覧上にフォーム表示 ──→ 保存
                      ├──[編集]──→ 一覧上でインライン編集 ──→ 保存
                      └──[削除]──→ 確認ダイアログ ──→ 削除
```
