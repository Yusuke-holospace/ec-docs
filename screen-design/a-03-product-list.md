# 画面設計書：A-03 商品一覧

> ワイヤーフレーム: `wireframes/a-03-product-list.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-03-product-list.html)
> PC専用

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | A-03 |
| 画面名 | 商品一覧 |
| URL | `/admin/products` |
| 認証 | 必要（管理者） |

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル＋新規登録ボタン | 右上に「新規登録」ボタン |
| ツールバー | 検索・カテゴリフィルタ・公開ステータスフィルタ |
| 商品テーブル | 一覧表示 |
| ページネーション | ページ番号ナビ |

---

## 3. セクション詳細

### 3.1 ツールバー

| 要素 | 詳細 |
|------|------|
| 商品名検索 | テキスト入力。商品名で部分一致検索 |
| カテゴリフィルタ | プルダウン。「すべてのカテゴリ」＋各カテゴリ |
| ステータスフィルタ | プルダウン。「すべてのステータス / 公開中 / 非公開」 |
| 表示件数 | 右端に「XX件表示中」 |

### 3.2 商品テーブル

| 列 | 内容 | 設定元 |
|----|------|--------|
| 商品 | サムネイル＋商品名＋カテゴリ名 | 商品別 |
| カテゴリ | カテゴリ名 | 商品別 |
| 最小ロット | 最小注文数量 | 商品別 |
| 単価（最小ロット時） | 価格テーブルの最小ロット時単価（C-04で定価として表示される値） | 商品別 |
| 公開/非公開 | トグルスイッチで即時切替 | 商品別 |
| 操作 | 「編集」→ A-05、「削除」→ 確認ダイアログ後に削除 | — |

### 3.3 ページネーション

| 要素 | 詳細 |
|------|------|
| 表示件数 | 1ページあたり20件 |
| ナビ | ページ番号＋「次へ」 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 「新規登録」クリック | A-04（商品登録）へ遷移 |
| 「編集」クリック | A-05（商品編集）へ遷移 |
| 「削除」クリック | 確認ダイアログ表示。確認後に削除 |
| 公開トグル切替 | 即時に公開/非公開を切替。購入者画面に即時反映 |
| 検索・フィルタ | 入力・選択で即時絞り込み |
| 行クリック | A-05（商品編集）へ遷移 |
