# 画面設計書：A-10 FAQ管理

> ワイヤーフレーム: `wireframes/a-10-faq.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-10-faq.html)
> PC専用

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | A-10 |
| 画面名 | FAQ管理 |
| URL | `/admin/faq` |
| 認証 | 必要（管理者） |

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル | 「FAQ管理」 |
| タブ切替 | 「サイトFAQ」/「グローバル商品FAQ」 |
| FAQカテゴリグループ | カテゴリ別にFAQを表示 |
| インラインフォーム | FAQ追加・編集用フォーム |

---

## 3. セクション詳細

### 3.1 タブ切替

| タブ | 詳細 | 管理ID |
|------|------|--------|
| サイトFAQ | C-15（FAQページ）に表示されるFAQ | ADM-007 |
| グローバル商品FAQ | 全商品のC-03（商品詳細ページ）に表示されるFAQ | ADM-008 |

### 3.2 FAQカテゴリグループ

| 要素 | 詳細 |
|------|------|
| カテゴリヘッダー | カテゴリ名＋アイコン＋FAQ件数＋「FAQ追加」ボタン |
| FAQアイテム | 並び順番号、質問、回答、操作リンク（編集・削除） |

### 3.3 FAQアイテム

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 並び順 | カテゴリ内の表示順（数値） | 管理者 |
| カテゴリ | 所属カテゴリ | 管理者 |
| 質問 | FAQ質問文 | 管理者 |
| 回答 | FAQ回答文 | 管理者 |

### 3.4 インライン追加・編集フォーム

| フィールド | 詳細 |
|------------|------|
| カテゴリ | プルダウン。既存カテゴリ＋「新規カテゴリ」 |
| 並び順 | 数値入力 |
| 質問 | テキスト入力 |
| 回答 | テキストエリア |
| ボタン | 「保存」「キャンセル」 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| タブ切替 | サイトFAQ / グローバル商品FAQの表示切替 |
| 「FAQ追加」クリック | カテゴリ配下にインライン追加フォームを表示 |
| 「編集」クリック | 該当FAQをインライン編集モードに切替 |
| 「削除」クリック | 確認ダイアログ表示。確認後に削除 |
| 「保存」クリック | FAQ内容を保存 |
| 「キャンセル」クリック | 編集・追加フォームを閉じる |
