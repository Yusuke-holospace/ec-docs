# 画面設計書：A-09 会員詳細

> ワイヤーフレーム: `wireframes/a-09-member-detail.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-09-member-detail.html)
> PC専用

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | A-09 |
| 画面名 | 会員詳細 |
| URL | `/admin/members/[id]` |
| 認証 | 必要（管理者） |

---

## 2. セクション構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル＋戻るリンク | 「会員詳細」＋「会員一覧に戻る」 |
| お客様情報 | 会社名、お名前、メール、電話番号、ログイン方法、登録日 |
| 住所 | 登録住所 |
| 配送先住所 | 配送先住所（登録住所と異なる場合） |
| 注文履歴 | この会員の注文一覧テーブル |

---

## 3. セクション詳細

### 3.1 お客様情報

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 会社名 | 法人の場合は会社名＋法人バッジ。個人の場合は非表示 | サーバー |
| お名前 | — | サーバー |
| メールアドレス | — | サーバー |
| 電話番号 | — | サーバー |
| ログイン方法 | マジックリンク / Google / Apple / Facebook / X | サーバー |
| 登録日 | YYYY/MM/DD | サーバー |

### 3.2 住所

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 住所 | 郵便番号＋住所 | サーバー |

### 3.3 配送先住所

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 会社名 | 任意（未入力の場合は非表示） | サーバー |
| お名前 | — | サーバー |
| 住所 | 郵便番号＋住所 | サーバー |
| 電話番号 | — | サーバー |

### 3.4 注文履歴

| 列 | 内容 | 設定元 |
|----|------|--------|
| 注文番号 | ORD-XXXXXXXX-XXXX | サーバー |
| チャネル | EC / 電話 / メール / 営業 | サーバー |
| ステータス | 現在のステータス（色付きバッジ） | サーバー |
| 商品数 | 注文内の商品点数 | サーバー |
| 合計金額 | 税込合計 | サーバー |
| 注文日 | YYYY/MM/DD | サーバー |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 「会員一覧に戻る」クリック | A-08（会員一覧）へ遷移 |
| 注文履歴の行クリック | A-07（注文詳細）へ遷移 |
