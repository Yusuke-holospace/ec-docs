## 1. 画面概要

|------|------|
| 画面ID | C-04 |
| 画面名 | カートページ |
| URL | `/cart` |
| 認証 | 不要（カートの閲覧・数量変更・削除はログイン不要。入稿・注文確定時にログイン必要） |

## 2. 画面構成

### PC版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | C-03と同じ2段構成 |
| パンくずリスト | トップ > カート |
| ページタイトル | 「カート（N点）」 |
| カート商品一覧（左） | テーブル形式。画像・商品情報・入稿状態・金額・数量・削除 |
| 注文サマリー（右・sticky） | 商品合計・送料・合計金額・注意事項・未入稿警告・ボタン・見積書DL |
| 空カート表示 | 商品0件の場合に表示 |
| パンくずリスト（下部） | ヘッダー下と同じ。フッターとの隙間なし |
| フッター（共通） | C-03と同じ |

### SP版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | C-03 SP版と同じ（ロゴ＋ログイン/カート/メニュー＋検索バー） |
| ページタイトル | 「カート（N点）」 |
| カート商品一覧 | カード型。画像＋情報＋入稿状態＋数量＋金額＋削除 |
| 注文サマリー | カード形式。警告＋ボタンを含む（下部固定バーなし） |
| 見積書DLリンク | サブリンク |
| フッター（共通） | C-03 SP版と同じ |

**SP版でのPC版との主な差分：**
- 2カラム → 1カラム（カード型縦積み）
- テーブルヘッダー非表示
- サマリーのstickyなし（通常のスクロール内に配置）
- 下部CTAバーなし（サマリー内にボタン配置）
- パンくず非表示

## 3. セクション詳細

### 3.1 ヘッダー・フッター（共通）

C-03画面設計書を参照。PC版・SP版ともにC-03と同じ共通コンポーネント。

### 3.2 カート商品一覧

#### テーブルヘッダー（PC版のみ）

| 列 | 内容 | 配置 |
|----|------|------|
| （画像） | ヘッダーなし | — |
| 商品 | 商品名・オプション | 左寄せ |
| 入稿状態 | 入稿済み / 未入稿 | 中央 |
| 金額 | 小計 | 中央 |
| 数量 | 数量入力 | 中央 |
| 削除 | 削除ボタン | 中央 |

#### 各商品行

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 商品画像 | 商品のサムネイル画像 | 商品別 |
| 注文種別バッジ | 「通常注文」（青）/ 「サンプル制作」（オレンジ） | カートデータ |
| 商品名 | リンク。クリックでC-03（商品詳細ページ）へ遷移 | 商品別 |
| オプション | 選択済みオプションを表示（例: サイズ: 44mm / 仕上げ: マット） | カートデータ |
| 入稿状態 | 「入稿済み」（緑背景）または「未入稿」（オレンジ背景） | カートデータ |
| 入稿リンク | 入稿済み:「データを変更」/ 未入稿:「入稿する」。クリックでC-05（入稿ページ）へ遷移。未ログイン時はC-08（ログイン）へ先に遷移 | — |
| 金額（小計） | 単価系合計 × 数量 ＋ 固定費。税込 | 自動算出 |
| 単価×数量 | 小計の内訳（例: @85円 × 100個） | 自動算出 |
| 固定費 | 「固定費: なし」または「固定費: +3,000円（色校正）」のように商品ごとに表示 | カートデータ |
| 数量入力（通常注文） | 数値入力フィールド。変更で価格が即時再計算 | — |
| 数量入力（サンプル） | 式数入力（1式=10個単位）。変更で価格が即時再計算 | — |
| 削除ボタン | クリックで商品をカートから削除 | — |

### 3.3 注文サマリー（PC: 右カラムsticky / SP: 通常スクロール内）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| タイトル | 「注文サマリー」 | 固定テキスト |
| 商品合計 | 全商品の小計合計（固定費含む）。「商品合計（N点）」と点数も表示 | 自動算出 |
| 送料 | カートの合計金額に応じて「無料」または金額を表示 | 共通（サイト設定の閾値で判定） |
| 合計（税込） | 商品合計 ＋ 送料 | 自動算出 |
| 注意事項 | 「※ 注文確定後、入稿データの確認を行います」等 | 固定テキスト |
| 未入稿警告 | 未入稿商品がある場合に表示。「未入稿の商品がN点あります。全商品の入稿を完了してください。」 | 自動判定 |
| 「注文に進む」ボタン | 全商品の入稿が完了している場合のみ活性。未入稿がある場合は非活性（グレー） | — |
| 「買い物を続ける」ボタン | C-02（商品一覧ページ）へ遷移 | 固定テキスト |
| 見積書ダウンロード | カート内全商品の見積書をPDFでダウンロード | — |

### 3.4 空カート表示

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| アイコン | プレースホルダー | — |
| メッセージ | 「カートに商品がありません」 | 固定テキスト |
| 「商品を探す」ボタン | C-02（商品一覧ページ）へ遷移 | 固定テキスト |

> カート内の商品が0件の場合、カート商品一覧とサマリーの代わりにこの表示を出す

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 数量変更（通常注文） | 商品の小計が即時再計算。サマリーの合計・送料も連動更新 |
| 式数変更（サンプル） | 式数 × 1式価格で小計を再計算。サマリーも連動更新 |
| 商品削除 | カートから商品を削除。ヘッダーのカートバッジ数も更新。最後の商品を削除したら空カート表示に切替 |
| 「入稿する」クリック | 未ログイン → C-08（ログイン）→ C-05（入稿ページ）。ログイン済み → C-05へ直接遷移 |
| 「データを変更」クリック | C-05（入稿ページ）へ遷移（再入稿モード） |
| 「注文に進む」クリック | C-06（注文確認ページ）へ遷移。全商品入稿完了時のみ活性 |
| 「買い物を続ける」クリック | C-02（商品一覧ページ）へ遷移 |
| 見積書DLクリック | カート内全商品の見積書PDFを生成・ダウンロード |
| 商品名クリック | C-03（該当商品の商品詳細ページ）へ遷移 |

---

## 5. バリデーション・エラー

| 条件 | 表示 |
|------|------|
| 通常注文で最小ロット未満 | 数量入力の下に赤字で「最小XX個」と表示 |
| サンプルで0式以下 | 最小1式のバリデーション |
| 全商品未入稿 or 一部未入稿 | サマリー内に警告メッセージ表示。「注文に進む」ボタンを非活性化 |

---

## 6. データ要件

| データ | 内容 | 設定元 |
|--------|------|--------|
| カート内商品リスト | 商品ID、オプション選択、数量/式数、注文種別（通常/サンプル） | カートデータ（未ログイン: ブラウザローカル、ログイン済み: サーバー） |
| 各商品の入稿状態 | 入稿済み / 未入稿 | サーバー（ログイン時のみ） |
| 商品情報 | 商品名、画像、オプション名、単価、固定費 | 商品別 |
| 送料閾値 | N円以上で送料無料 | 共通（サイト設定） |
| 送料金額 | 閾値未満の場合の送料 | 共通（サイト設定） |
