## 2. 画面構成

### PC版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | C-03と同じ2段構成 |
| パンくずリスト | トップ > カート |
| 進捗バー | 4ステップ: カート(active) → データ入稿 → 注文確認 → 完了。C-05〜C-07と共通 |
| ページタイトル | 「カート（N点）」 |
| カート商品一覧（左） | 商品名バー（黒背景）＋テーブル行。列: 画像・商品仕様・入稿状態・単価・数量・小計 |
| 注文サマリー（右・sticky） | 商品合計・送料・合計金額・注意事項・未入稿警告・ボタン・見積書DL |
| 空カート表示 | 商品0件の場合に表示 |
| パンくずリスト（下部） | ヘッダー下と同じ。フッターとの隙間なし |
| フッター（共通） | C-03と同じ |

### SP版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | C-03 SP版と同じ（ロゴ＋ログイン/カート/メニュー＋検索バー） |
| 進捗バー | 4ステップ: カート(active) → データ入稿 → 注文確認 → 完了 |
| ページタイトル | 「カート（N点）」 |
| カート商品一覧 | 商品名バー（黒背景）＋カード型。3列グリッド: 入稿状態・単価・数量。小計行あり |
| 注文サマリー | カード形式。警告＋ボタンを含む（下部固定バーなし） |
| 見積書DLリンク | サブリンク |
| フッター（共通） | C-03 SP版と同じ |

**SP版でのPC版との主な差分：**
- 2カラム → 1カラム（カード型縦積み）
- テーブルヘッダー非表示。代わりに各項目にラベル（入稿状態・単価・数量）を表示
- 商品行内を3列グリッド（入稿状態 / 単価+固定費 / 数量）で配置
- 小計は行の下部に右寄せで表示
- サマリーのstickyなし（通常のスクロール内に配置）
- 下部CTAバーなし（サマリー内にボタン配置）
- パンくず非表示

## 3. セクション詳細

### 3.1 ヘッダー・フッター（共通）

C-03画面設計書を参照。PC版・SP版ともにC-03と同じ共通コンポーネント。

### 3.2 カート商品一覧

#### 商品名バー（各商品の上部）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 背景色 | #333（黒） | — |
| 注文種別バッジ | 「通常注文」（青）/ 「サンプル制作」（オレンジ） | カートデータ |
| 商品名 | 白文字リンク。クリックでC-03（商品詳細ページ）へ遷移 | 商品別 |
| 「複製」ボタン | 同じ商品・オプション・数量でカートにもう1行追加 | — |
| 「削除」ボタン | 確認ダイアログなしで即時削除 | — |

#### テーブルヘッダー（PC版のみ）

| 列 | 内容 | 配置 |
|----|------|------|
| （画像） | ヘッダーなし | — |
| 商品・仕様 | オプション情報 | 左寄せ |
| 入稿状態 | 入稿済み / 未入稿 | 中央 |
| 単価 | 単価＋固定費 | 中央 |
| 数量 | 数量入力 | 中央 |
| 小計 | 小計額 | 右寄せ |

#### 各商品行

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 商品画像 | 商品のサムネイル画像 | 商品別 |
| オプション（商品・仕様列） | 選択済みオプションを改行なしで表示（例: サイズ: 44mm / 仕上げ: マット）。黒字・13px | カートデータ |
| 入稿状態 | 「入稿済み」（緑背景）または「未入稿」（オレンジ背景） | カートデータ |
| 入稿リンク | 入稿状態の下に表示。入稿済み:「データを変更」/ 未入稿:「入稿する」。クリックでC-05（入稿ページ）へ遷移。未ログイン時はC-08（ログイン）へ先に遷移 | — |
| 単価（割引後） | 数量割引適用後の単価 | 自動算出（価格テーブルから） |
| 単価（定価） | 取消線付きで定価を表示 | 商品別 |
| 固定費 | 「固定費: なし」または「固定費: +¥3,000」のように単価列の下に表示 | カートデータ |
| 数量入力 | 数値入力フィールド。「個」単位。変更で小計・サマリーが即時再計算 | — |
| 小計額 | 単価 × 数量 ＋ 固定費。税込 | 自動算出 |
| 小計定価 | 取消線付きで定価ベースの合計を表示（数量割引がある場合のみ） | 自動算出 |
| 「数量割引」ラベル | 割引が適用されている場合に小計の下に赤字で表示 | 自動判定 |

### 3.3 注文サマリー（PC: 右カラムsticky / SP: 通常スクロール内）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| タイトル | 「注文サマリー」 | 固定テキスト |
| 商品合計 | 全商品の小計合計（固定費含む）。「商品合計（N点）」と点数も表示 | 自動算出 |
| 送料 | 各商品の送料を合算して表示。カート合計金額がN円以上なら「無料」 | 商品別（各商品に送料設定）＋共通（無料閾値） |
| 納期目安 | カート内全商品の中で最も長い納期を表示。「約X営業日」形式（単一値）。営業日ベース（土日祝除く） | 商品別（商品・数量に応じた納期設定） |
| 合計（税込） | 商品合計 ＋ 送料 | 自動算出 |
| 注意事項 | 「※ 注文確定後、入稿データの確認を行います」等 | 固定テキスト |
| 未入稿警告 | 未入稿商品がある場合に表示。「未入稿の商品がN点あります。全商品の入稿を完了してください。」 | 自動判定 |
| 「注文に進む」ボタン | 全商品の入稿が完了している場合のみ活性。未入稿がある場合は非活性（グレー） | — |
| 「買い物を続ける」ボタン | C-02（商品一覧ページ）へ遷移 | 固定テキスト |
| 見積書ダウンロード | カート内全商品の見積書をPDFでダウンロード | — |

### 3.4 空カート表示

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| アイコン | プレースホルダー | — |
| メッセージ | 「カートに商品がありません」 | 固定テキスト |
| 「商品を探す」ボタン | C-02（商品一覧ページ）へ遷移 | 固定テキスト |

> カート内の商品が0件の場合、カート商品一覧とサマリーの代わりにこの表示を出す

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 数量変更 | 商品の小計が即時再計算。サマリーの合計・送料も連動更新 |
| 商品削除 | 商品名バーの「削除」ボタンから実行。確認ダイアログなしで即時削除。ヘッダーのカートバッジ数も更新。最後の商品を削除したら空カート表示に切替 |
| 商品複製 | 商品名バーの「複製」ボタンから実行。同じ商品・オプション・数量でカートにもう1行追加 |
| 「入稿する」クリック | 未ログイン → C-08（ログイン）→ C-05（入稿ページ）。ログイン済み → C-05へ直接遷移 |
| 「データを変更」クリック | C-05（入稿ページ）へ遷移（再入稿モード） |
| 「注文に進む」クリック | C-06（注文確認ページ）へ遷移。全商品入稿完了時のみ活性 |
| 「買い物を続ける」クリック | C-02（商品一覧ページ）へ遷移 |
| 見積書DLクリック | カート内全商品の見積書PDFを生成・ダウンロード |
| 商品名クリック | C-03（該当商品の商品詳細ページ）へ遷移 |

---

## 5. バリデーション・エラー

| 条件 | 表示 |
|------|------|
| 通常注文で最小ロット未満 | 数量入力の下に赤字で「最小XX個」と表示 |
| サンプルで0個以下 | 最小1個のバリデーション |
| 全商品未入稿 or 一部未入稿 | サマリー内に警告メッセージ表示。「注文に進む」ボタンを非活性化 |

---

## 6. データ要件

| データ | 内容 | 設定元 |
|--------|------|--------|
| カート内商品リスト | 商品ID、オプション選択、数量、注文種別（通常/サンプル） | カートデータ（未ログイン: ブラウザローカル、ログイン済み: サーバー） |
| 各商品の入稿状態 | 入稿済み / 未入稿 | サーバー（ログイン時のみ） |
| 商品情報 | 商品名、画像、オプション名、単価（定価）、価格テーブル（数量別単価）、固定費 | 商品別 |
| 商品別送料 | 各商品に設定された送料 | 商品別 |
| 送料無料閾値 | N円以上で送料無料 | 共通（サイト設定） |
