# 画面設計書：A-14 代理注文作成

> ワイヤーフレーム: `wireframes/a-14-proxy-order.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-14-proxy-order.html)
> PC専用

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | A-14 |
| 画面名 | 代理注文作成 |
| URL | `/admin/orders/new` |
| 認証 | 必要（管理者） |

---

## 2. セクション構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル＋戻るリンク | 「代理注文作成」＋「注文一覧に戻る」 |
| 1. 顧客選択 | 既存顧客検索 or 新規顧客登録 |
| 2. 商品追加 | 商品検索・オプション選択・数量・入稿データ |
| 3. 連絡先 | メール・電話番号 |
| 4. チャネル・支払い方法 | 注文チャネル＋支払い方法選択 |
| 5. 備考 | テキストエリア |
| 送信バー | 商品合計表示＋「注文を作成」ボタン |

---

## 3. セクション詳細

### 3.1 顧客選択

| 要素 | 詳細 |
|------|------|
| 顧客検索 | 名前・会社名・メールで既存顧客を検索 |
| 検索結果 | 名前・会社名・メールを表示。クリックで選択 |
| 選択済み表示 | 青枠で選択済み顧客を表示。「変更」リンクで再選択可能 |
| 新規顧客登録 | 「または」区切りで新規顧客フォームを表示 |

#### 新規顧客フォーム

| フィールド | 必須 | 詳細 |
|------------|------|------|
| お名前 | 必須 | テキスト入力 |
| 会社名 | 任意 | テキスト入力 |
| メールアドレス | 必須 | メール入力 |
| 電話番号 | 必須 | 電話番号入力 |

### 3.2 商品追加

| 要素 | 詳細 |
|------|------|
| 商品検索 | 商品名で検索。結果から選択して追加 |
| 追加済み商品 | 商品ごとにオプション選択（プルダウン）・数量入力・小計表示 |
| 入稿データ | 商品ごとにファイルアップロード（任意） |
| 削除ボタン | 追加済み商品を削除 |

#### 追加済み商品アイテム

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 商品サムネイル | 商品画像 | サーバー |
| 商品名 | 商品名称 | サーバー |
| オプション | 商品ごとのオプション（サイズ・仕上げ・付属等）をプルダウンで選択 | サーバー（選択肢）/ 管理者（選択値） |
| 数量 | 数値入力 | 管理者 |
| 入稿データ | ファイルアップロード（任意）。アップロード済みの場合はファイル名とサイズを表示 | 管理者 |
| 小計 | 単価 x 数量 | サーバー（自動計算） |

### 3.3 連絡先

| フィールド | 詳細 | 設定元 |
|------------|------|--------|
| メールアドレス | 顧客選択時は自動入力。変更可能 | サーバー / 管理者 |
| 電話番号 | 顧客選択時は自動入力。変更可能 | サーバー / 管理者 |
| 注記 | 「この注文に関する連絡先を指定します（顧客の登録情報と異なる場合）」 | 固定テキスト |

### 3.4 チャネル・支払い方法

| フィールド | 詳細 |
|------------|------|
| 注文チャネル | プルダウン。「電話 / メール / 営業」 |
| 支払い方法 | プルダウン。「クレジットカード / 銀行振込 / 請求書払い」 |

### 3.5 備考

| フィールド | 詳細 |
|------------|------|
| 備考 | テキストエリア。社内メモ・特記事項等 |

### 3.6 送信バー

| 要素 | 詳細 |
|------|------|
| 商品合計 | 商品点数＋合計金額を表示 |
| 「下書き保存」ボタン | 注文を下書き状態で保存 |
| 「注文を作成」ボタン | 注文を確定して作成 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 顧客検索 | 名前・会社名・メールで既存顧客を検索。結果をリスト表示 |
| 顧客選択 | クリックで顧客を選択。連絡先に自動入力 |
| 「変更」クリック | 選択済み顧客を解除して再検索可能に |
| 商品検索 | 商品名で検索。結果から選択して追加 |
| オプション・数量変更 | 小計を即時再計算 |
| 入稿データアップロード | 商品ごとにファイルをアップロード（任意） |
| 商品削除 | 追加済み商品を削除。合計を再計算 |
| 「注文を作成」クリック | 注文を作成。A-07（注文詳細）へ遷移。共有URLが自動発行される |
| 「下書き保存」クリック | 注文を下書き状態で保存 |
| 「注文一覧に戻る」クリック | A-06（注文一覧）へ遷移 |
