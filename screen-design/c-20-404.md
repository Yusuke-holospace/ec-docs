## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-20 |
| 画面名 | 404エラーページ |
| URL | 存在しないURLにアクセスした場合に表示 |
| 認証 | 不要 |
| 対応デバイス | PC・スマートフォン（レスポンシブ） |

## 2. セクション詳細

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| エラーコード | 「404」大きめフォントで表示 | 固定テキスト |
| タイトル | 「ページが見つかりません」 | 固定テキスト |
| 説明文 | 「お探しのページは存在しないか、移動した可能性があります。」 | 固定テキスト |
| 「トップページへ」ボタン | C-01へ遷移 | 固定テキスト |
| 「商品一覧へ」ボタン | C-02へ遷移 | 固定テキスト |
