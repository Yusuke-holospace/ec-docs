## 2. セクション詳細

### 2.1 ヒーローバナー（スライダー）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| スライド枚数 | 3枚（管理画面で追加・編集・並び替え可能にするかは要検討） | 共通 |
| 自動再生 | 5秒間隔で次スライドへ | — |
| ドットインジケーター | スライド下部に表示。クリックで該当スライドへ移動 | — |
| 各スライド構成 | 背景画像 + キャッチコピー + サブコピー + CTAボタン | スライド別 |

**スライド内容（初期設定）：**

| # | キャッチコピー | サブコピー | CTA | 遷移先 |
|---|--------------|-----------|-----|--------|
| 1 | 推し活グッズを、もっと手軽に。 | デザインデータを入稿するだけ。オリジナルグッズの小ロット制作から大量生産まで対応。 | 商品を探す | C-02 |
| 2 | サンプル10個から制作OK | まずは少量から試せる。色味やサイズ感を実物で確認。 | サンプル制作について | C-16（ご利用ガイド） |
| 3 | テンプレートで簡単入稿 | 商品ごとのテンプレートをダウンロード。デザインを当てはめてアップロードするだけ。 | ご利用ガイドを見る | C-16 |

> PC版のスライド1はサブCTA「ご利用ガイドを見る」も表示

### 2.2 商品カテゴリ一覧

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| セクションタイトル | なし（無印良品を参考にシンプルな構成） | — |
| レイアウト | PC: 6列×2行グリッド / SP: 3列×4行グリッド | — |
| カード構成 | カテゴリ画像（正方形）+ カテゴリ名のみ。価格非掲載。ボーダーなし、背景はページ背景と同色 | カテゴリ別 |
| クリック | C-02（商品一覧ページ）にカテゴリフィルタ付きで遷移 | — |

**表示カテゴリ（12個）：**
缶バッジ / アクリルスタンド / キーホルダー / ストラップ / トレーディングカード / クリアファイル / ステッカー・シール / タオル・布製品 / Tシャツ / トートバッグ / うちわ / その他グッズ

> カテゴリは実際の商品ラインナップ確定後に調整。

### 2.6 ご利用の流れ

| 要素 | 詳細 |
|------|------|
| セクションタイトル | 「ご利用の流れ」 |
| レイアウト | PC: 4列横並び（ステップ間に→矢印）/ SP: 縦積み（ステップ間に↓矢印） |
| 各ステップ | 番号 + アイコン（PC版のみ）+ タイトル + 説明文 |
| 「ご利用ガイドへ」リンク | C-16（ご利用ガイド）へ遷移 |

**ステップ内容：**

| # | タイトル | 説明 |
|---|---------|------|
| 1 | 商品を選ぶ | カテゴリや検索から商品を選択。オプション・数量を指定してカートに追加。 |
| 2 | デザインデータを入稿 | テンプレートをダウンロードし、デザインを当てはめてアップロード。 |
| 3 | スタッフが確認・決済案内 | 入稿データを専門スタッフが確認。問題なければ決済のご案内をお送りします。 |
| 4 | 製造・お届け | お支払い確認後に製造を開始。完了次第、発送いたします。 |

### 2.7 選ばれる理由

| 要素 | 詳細 |
|------|------|
| セクションタイトル | 「選ばれる理由」 |
| レイアウト | PC: 3列グリッド / SP: 縦積みリスト |
| 各カード | アイコン + タイトル + 説明文 |

**内容：**

| タイトル | 説明 |
|---------|------|
| 小ロットOK | サンプル制作は10個から対応。まずは少量で試して、納得してから量産できます。 |
| かんたん入稿 | 商品ごとにテンプレートをご用意。ダウンロードしてデザインを当てはめるだけ。 |
| 安心のデータ確認 | 入稿データは専門スタッフが1点ずつ確認。不備があれば製造前にご連絡します。 |

## 3. PC/SP差分

| セクション | PC | SP |
|-----------|----|----|
| ヒーロー | 高さ420px、サブCTAあり | 高さ240px、サブCTAなし |
| カテゴリ | 6列×2行グリッド | 3列×4行グリッド |
| ランキング | 4列グリッド | 横スクロール（カード幅150px） |
| 新商品 | 4列グリッド | 横スクロール（カード幅150px） |
| 価格帯 | 4列グリッド | 横スクロール（カード幅150px） |
| 利用の流れ | 横並び4列（→矢印） | 縦積み（↓矢印）、アイコンなし |
| 選ばれる理由 | 3列グリッド（中央揃え） | 縦積みリスト（横並びアイコン+テキスト） |
| 制作事例 | 3列グリッド | 横スクロール（カード幅220px） |
| CTA | ボタン横並び | ボタン縦積み |

## 4. 補足

- トップページにはパンくずリストを表示しない
- ヘッダーのログインリンクはログイン状態で「マイページ」に切替（他画面と共通仕様）
- 価格はすべて税込表示
- ランキング・新商品・価格帯の商品データは動的に取得（商品マスタから）
- 新商品にはサムネイル左上に赤い「NEW」バッジを表示
- 制作事例は静的コンテンツ。専用一覧ページは設けない
