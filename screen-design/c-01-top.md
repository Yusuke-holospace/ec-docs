## 2. セクション詳細

### 2.1 ヒーローカルーセル

無印良品を参考にしたカルーセル形式。正方形のバナー画像が中央に表示され、左右に前後のスライドが2/3程度見える。

| 要素 | 詳細 |
|------|------|
| スライド枚数 | 5枚（管理画面で追加・編集・並び替え可能にするかは要検討） |
| スライド構成 | バナー画像のみ（正方形）。テキスト・CTAはバナー画像内で処理 |
| スライド幅 | 40vw（画面幅に応じて伸縮） |
| 左右表示 | 前後のスライドが2/3程度見える。非アクティブスライドはopacity: 0.5 |
| ループ | 無限ループ（左右どちらにも途切れず回る） |
| 自動再生 | 5秒間隔で次スライドへ |
| ナビゲーション | `<` `>` 矢印ボタン（Lucide: `chevron-left` / `chevron-right`）+ ドットインジケーター |
| トランジション | transform 0.5s ease |
| クリック | バナー画像ごとに指定したURLに遷移（主にC-03 商品詳細ページ）。リンク先は管理画面でバナーごとに設定 |

### 2.2 商品カテゴリ一覧

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| セクションタイトル | なし（無印良品を参考にシンプルな構成） | — |
| レイアウト | PC: 6列×2行グリッド / SP: 3列×4行グリッド | — |
| カード構成 | カテゴリ画像（正方形）+ カテゴリ名のみ。価格非掲載。ボーダーなし、背景はページ背景と同色 | カテゴリ別 |
| クリック | C-02（商品一覧ページ）にカテゴリフィルタ付きで遷移 | — |

**表示カテゴリ（12個）：**
缶バッジ / アクリルスタンド / キーホルダー / ストラップ / トレーディングカード / クリアファイル / ステッカー・シール / タオル・布製品 / Tシャツ / トートバッグ / うちわ / その他グッズ

> カテゴリは実際の商品ラインナップ確定後に調整。

### 2.3 人気商品

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| セクションタイトル | 「人気商品」 | — |
| レイアウト | PC: 4列グリッド / SP: 横スクロール | — |
| 表示件数 | 4件 | — |
| 選定方法 | 管理者が手動で選んだ商品を表示（自動集計ではない）。管理画面で選択・並び順を設定 | 管理者設定 |
| タグ | 各商品に1つタグを設定可能（例: 「人気No.1」「おすすめ」「定番」等）。管理画面で自由入力 | 管理者設定 |
| カード構成 | タグ + 商品サムネイル（ホバーで①→②切り替え）+ カテゴリ名 + 商品名 + 単価 | 商品別 |
| 単価表記 | ロット数10,000個時の単価を表示。形式: 「@XX円（税込）〜」 | 商品別 |
| クリック | C-03（商品詳細ページ）へ遷移 | — |

### 2.6 ご利用の流れ

| 要素 | 詳細 |
|------|------|
| セクションタイトル | 「ご利用の流れ」 |
| レイアウト | PC: 4列横並び（ステップ間に→矢印）/ SP: 縦積み（ステップ間に↓矢印） |
| 各ステップ | 番号 + 画像プレースホルダー（PC版のみ、180×180px、「ご利用の流れ画像1〜4」）+ タイトル + 説明文 |
| 矢印位置 | PC: 画像の高さ中心に配置 / SP: ステップ間に↓ |
| 「ご利用ガイドへ」リンク | C-16（ご利用ガイド）へ遷移 |

**ステップ内容：**

| # | タイトル | 説明 |
|---|---------|------|
| 1 | 商品を選ぶ | カテゴリや検索から商品を選択。オプション・数量を指定してカートに追加。 |
| 2 | デザインデータを入稿 | テンプレートをダウンロードし、デザインを当てはめてアップロード。 |
| 3 | スタッフが確認・決済案内 | 入稿データを専門スタッフが確認。問題なければ決済のご案内をお送りします。 |
| 4 | 製造・お届け | お支払い確認後に製造を開始。完了次第、発送いたします。 |

### 2.7 選ばれる理由

| 要素 | 詳細 |
|------|------|
| セクションタイトル | 「選ばれる理由」 |
| レイアウト | PC: 3列グリッド / SP: 縦積みリスト |
| 各カード | PC: 画像プレースホルダー（140×140px、「選ばれる理由画像1〜3」）+ タイトル + 説明文 / SP: タイトル + 説明文のみ（画像なし） |

**内容：**

| タイトル | 説明 |
|---------|------|
| 小ロットOK | サンプル制作は10個から対応。まずは少量で試して、納得してから量産できます。 |
| かんたん入稿 | 商品ごとにテンプレートをご用意。ダウンロードしてデザインを当てはめるだけ。 |
| 安心のデータ確認 | 入稿データは専門スタッフが1点ずつ確認。不備があれば製造前にご連絡します。 |
