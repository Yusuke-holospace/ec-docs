# 画面設計書：A-08 会員一覧

> ワイヤーフレーム: `wireframes/a-08-member-list.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-08-member-list.html)
> PC専用

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | A-08 |
| 画面名 | 会員一覧 |
| URL | `/admin/members` |
| 認証 | 必要（管理者） |

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル | 「会員一覧」 |
| ツールバー | 検索・法人/個人フィルタ |
| 会員テーブル | 一覧表示 |
| ページネーション | ページ番号ナビ |

---

## 3. セクション詳細

### 3.1 ツールバー

| 要素 | 詳細 |
|------|------|
| 検索 | テキスト入力。名前・会社名・メールアドレスで部分一致検索 |
| 法人/個人フィルタ | プルダウン。「すべて / 法人 / 個人」 |
| 表示件数 | 右端に「XX件表示中」 |

### 3.2 会員テーブル

| 列 | 内容 | 設定元 |
|----|------|--------|
| お名前 | 会員名＋法人/個人バッジ | サーバー |
| 会社名 | 法人の場合は会社名、個人の場合は「—」 | サーバー |
| メール | メールアドレス | サーバー |
| 電話番号 | 電話番号 | サーバー |
| 注文数 | この会員の総注文数 | サーバー |
| 登録日 | 会員登録日（YYYY/MM/DD） | サーバー |

### 3.3 ページネーション

| 要素 | 詳細 |
|------|------|
| 表示件数 | 1ページあたり20件 |
| ナビ | ページ番号＋「次へ」 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 検索入力 | 名前・会社名・メールで即時絞り込み |
| フィルタ選択 | 法人/個人で即時絞り込み |
| 行クリック | A-09（会員詳細）へ遷移 |
