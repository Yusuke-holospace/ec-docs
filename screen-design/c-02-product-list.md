## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-02 |
| 画面名 | 商品一覧ページ |
| URL | `/products`（全商品）/ `/products?category=xxx`（カテゴリ）/ `/products?q=xxx`（検索結果） |
| 認証 | 不要 |
| 対応デバイス | PC・スマートフォン（レスポンシブ） |

---

## 2. セクション詳細

### 2.1 パンくずリスト（PC版のみ）

| パターン | 表示 |
|---------|------|
| 全商品 | トップ > 全商品 |
| カテゴリ | トップ > {カテゴリ名} |
| 検索結果 | トップ > 「{キーワード}」の検索結果 |

### 2.2 ページタイトル

| 要素 | 詳細 |
|------|------|
| タイトル | カテゴリ名 or 「{キーワード}の検索結果」or 「全商品」 |

### 2.3 フィルタ・並び替え

フィルタバーは2行構成（PC版）。ヘッダーのカテゴリナビは「ページ遷移」、ページ内フィルタは「ページ内絞り込み」と役割が異なる。検索結果ページ（/products?q=xxx）でもカテゴリフィルタが利用可能。カテゴリはプルダウン（カテゴリ数が増えても崩れないため）、価格帯は選択肢が固定のためピル型を採用。

**PC版:**

| 行 | 要素 | 詳細 |
|----|------|------|
| 1行目 | カテゴリフィルタ + 価格帯フィルタ | カテゴリ: プルダウン / 価格帯: ピル型ボタン「すべて」「〜99円」「100〜199円」「200〜499円」「500円〜」 |
| 2行目 | 件数 + 並び替え | 左に「XX件の商品」、右にプルダウン |

**SP版:**

| 要素 | 詳細 |
|------|------|
| カテゴリフィルタ | プルダウン（select） |
| 価格帯フィルタ | プルダウン（select） |
| 並び替え | プルダウン（select） |
| 件数 | フィルタ下に「XX件の商品」テキスト |

**並び替えオプション:**
- おすすめ順（デフォルト。商品登録時に設定する「おすすめ度」5段階で降順。同じ段階なら登録日が新しい順）
- 新着順
- 価格が安い順
- 価格が高い順

### 2.4 商品グリッド

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| レイアウト | PC: 4列グリッド / SP: 2列グリッド | — |
| カード構成 | 商品画像（1:1）+ カテゴリラベル + 商品名 + 単価（@XX円〜・税込）+ 納期目安 | 商品別 |
| クリック | C-03（商品詳細ページ）へ遷移 | — |
| 0件時 | 「該当する商品が見つかりませんでした。」メッセージを表示 | — |

### 2.5 ページネーション

| 要素 | 詳細 |
|------|------|
| 表示形式 | PC: 「< 前へ」+ ページ番号 + 「...」+ 「次へ >」/ SP: 「<」+ ページ番号 + 「>」 |
| 1ページあたり | 20件（要検討） |
| 現在ページ | アクティブスタイル（黒背景白文字） |
| 先頭/末尾 | 「前へ」「次へ」が無効状態 |

## 3. PC/SP差分

| セクション | PC | SP |
|-----------|----|----|
| パンくず | ヘッダー下 + フッター上 | なし |
| カテゴリフィルタ | プルダウン（select） | プルダウン（select） |
| 価格帯フィルタ | ピル型ボタン横並び | プルダウン（select） |
| 並び替え | プルダウン | プルダウン |
| 件数表示 | フィルタバー2行目（左） | フィルタバー下 |
| 商品グリッド | 4列 | 2列 |
| ページネーション | 「前へ/次へ」テキスト付き | 「< / >」記号のみ |

---

## 4. 補足

- 検索結果ページも本画面で表示（URLパラメータ `q` で判定）
- カテゴリ選択時はURLパラメータ `category` を変更し、フィルタのアクティブ状態と連動
- 価格はすべて税込表示
- 商品データは商品マスタから動的に取得
- 並び替え・フィルタ変更時はページをリロードし、1ページ目に戻る（初期実装）。将来的にSPA化も検討可
- 単価はロット数10,000個時の単価を表示。形式: 「@XX円（税込）〜」
