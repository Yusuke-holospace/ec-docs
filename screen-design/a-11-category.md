# 画面設計書：A-11 カテゴリ管理

> ワイヤーフレーム: `wireframes/a-11-category.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-11-category.html)
> PC専用

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | A-11 |
| 画面名 | カテゴリ管理 |
| URL | `/admin/categories` |
| 認証 | 必要（管理者） |

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル＋新規追加ボタン | 右上に「新規追加」ボタン |
| カテゴリリスト | ドラッグ並び替え可能なリスト |
| 編集モーダル | カテゴリ編集フォーム |

---

## 3. セクション詳細

### 3.1 カテゴリリスト

| 列 | 内容 | 設定元 |
|----|------|--------|
| ドラッグハンドル | 並び順変更用のグリップアイコン | — |
| 画像 | 画像①（通常表示）と画像②（ホバー時切替） | 管理者 |
| カテゴリ名 | カテゴリ名称 | 管理者 |
| 説明 | カテゴリの説明文 | 管理者 |
| 商品数 | カテゴリに属する商品の件数 | サーバー |
| 並び順 | 現在の表示順番号 | 管理者 |
| 操作 | 「編集」「削除」 | — |

### 3.2 編集モーダル

| フィールド | 詳細 |
|------------|------|
| カテゴリ名 | テキスト入力 |
| 説明 | テキストエリア |
| カテゴリ画像①（通常表示） | 画像アップロード。C-02等でデフォルト表示される画像 |
| カテゴリ画像②（ホバー時） | 画像アップロード。ホバー時に切り替わる画像 |
| ボタン | 「保存」「キャンセル」 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 「新規追加」クリック | 空の編集モーダルを表示 |
| 「編集」クリック | 該当カテゴリの編集モーダルを表示 |
| 「削除」クリック | 確認ダイアログ表示。商品が存在する場合は警告。確認後に削除 |
| ドラッグ&ドロップ | 並び順を変更。変更は自動保存 |
| モーダル「保存」クリック | カテゴリ情報を保存してモーダルを閉じる |
| モーダル「キャンセル」クリック | 変更を破棄してモーダルを閉じる |
