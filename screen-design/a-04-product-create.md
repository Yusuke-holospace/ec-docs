# 画面設計書：A-04 商品登録 / A-05 商品編集

> ワイヤーフレーム: `wireframes/a-04-product-create.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-04-product-create.html)
> PC専用。A-05（商品編集）は同じレイアウトで既存データが入力済みの状態。

---

## 1. 画面概要

| 項目 | A-04 | A-05 |
|------|------|------|
| 画面名 | 商品登録 | 商品編集 |
| URL | `/admin/products/new` | `/admin/products/[id]/edit` |
| 認証 | 必要（管理者） | 必要（管理者） |

---

## 2. セクション構成

| セクション | 内容 |
|------------|------|
| 基本情報 | 商品名・カテゴリ・おすすめ度・説明・公開ステータス |
| 商品画像 | 複数画像アップロード・並び替え |
| オプション設定 | 動的に追加可能なオプション項目（項目名＋選択肢） |
| 価格テーブル | 数量帯ごとの単価＋納期（営業日） |
| 固定費・送料 | 固定費オプション（色校正等）＋送料 |
| サンプル制作 | サンプル受付有無＋単価 |
| 入稿設定 | 入稿データ要件＋スロット定義＋テンプレートファイル |
| 商品FAQ | この商品固有のFAQ |

---

## 3. セクション詳細

### 3.1 基本情報

| 項目 | 必須/任意 | 入力形式 | 備考 |
|------|----------|----------|------|
| 商品名 | 必須 | テキスト | — |
| カテゴリ | 必須 | プルダウン | A-11で管理するカテゴリから選択 |
| おすすめ度 | 任意 | 5段階（★） | C-01トップページのおすすめ商品に表示する選定基準。デフォルト0 |
| 商品説明 | 必須 | テキストエリア | C-03商品詳細に表示 |
| 公開ステータス | — | プルダウン | 「非公開（下書き）」「公開」。デフォルト非公開 |

### 3.2 商品画像

| 項目 | 詳細 |
|------|------|
| アップロード | 複数画像アップロード可能。ドラッグで並び替え |
| メイン画像 | 1枚目がメイン画像として一覧・カートに表示 |
| ホバー画像 | 2枚目がホバー時の切替画像（C-01カテゴリ、C-02一覧、C-03関連商品） |

### 3.3 オプション設定

| 項目 | 詳細 |
|------|------|
| 動的追加 | 「オプションを追加」で項目を増やせる |
| 各オプション | 項目名（テキスト）＋ 選択肢（カンマ区切りテキスト） |
| 削除 | ×ボタンで削除 |
| 例 | サイズ: 44mm, 56mm, 76mm / 仕上げ: マット, グロス |

### 3.4 価格テーブル

| 列 | 詳細 |
|----|------|
| 数量（個） | 数量帯の下限値。1行目が最小ロット |
| 単価（税込） | その数量帯の単価。最小ロット時の単価がC-04で定価として表示 |
| 納期（営業日） | その数量帯の納期目安。入金確認完了後から起算 |
| 行の追加/削除 | 動的に増減可能 |

### 3.5 固定費・送料

| 項目 | 詳細 |
|------|------|
| 固定費オプション | 動的に追加。費目名＋金額。例: 色校正 3,000円 |
| 送料 | 商品ごとの送料（税込）。サイト全体の送料無料閾値を超えた場合は自動無料 |

### 3.6 サンプル制作

| 項目 | 詳細 |
|------|------|
| 受付有無 | はい/いいえ。「いいえ」の場合C-03にサンプルボタン非表示 |
| サンプル単価 | サンプル1個あたりの価格（税込） |

### 3.7 入稿設定

#### 入稿データの要件

| 項目 | 詳細 |
|------|------|
| 対応形式 | テキスト入力（例: AI, PSD, PDF） |
| カラーモード | テキスト入力（例: CMYK） |
| 推奨解像度 | テキスト入力（例: 350dpi以上） |
| 注意事項 | テキスト入力（例: アウトライン化必須） |

#### 入稿スロット

| 項目 | 詳細 |
|------|------|
| 動的追加 | 「スロットを追加」で増やせる |
| ラベル | スロット名（例: 表面デザイン、裏面デザイン） |
| 必須/任意 | プルダウン |
| 表示条件 | 「常に表示」or「特定オプション選択時」。後者の場合、対象オプション＋値を指定 |
| テンプレートファイル | スロットごとにテンプレートファイルをアップロード。C-03・C-05からダウンロード可能 |

### 3.8 商品FAQ

| 項目 | 詳細 |
|------|------|
| 動的追加 | 「FAQを追加」で増やせる |
| 各FAQ | 質問（テキスト）＋ 回答（テキストエリア） |
| 削除 | ×ボタンで削除 |
| 表示先 | C-03商品詳細ページのFAQセクション。グローバルFAQ（ADM-008）とは別管理 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 「商品を登録する」クリック | バリデーション実行。成功 → 保存してA-03に遷移 |
| 「キャンセル」クリック | 変更を破棄してA-03に遷移 |
| 「商品一覧に戻る」クリック | A-03に遷移（変更がある場合は確認ダイアログ） |
| 画像ドラッグ | 画像の並び順を変更 |
| 各「追加」ボタン | オプション/数量帯/固定費/スロット/FAQを動的追加 |
| 各「×」ボタン | 該当項目を削除（確認ダイアログなし） |

---

## 5. バリデーション

| 条件 | 表示 |
|------|------|
| 商品名が未入力 | 「商品名を入力してください」 |
| カテゴリが未選択 | 「カテゴリを選択してください」 |
| 商品説明が未入力 | 「商品説明を入力してください」 |
| 価格テーブルが0行 | 「価格テーブルを1行以上設定してください」 |
| 入稿スロットが0件 | 「入稿スロットを1件以上設定してください」 |
