# 画面設計書：A-04 商品登録 / A-05 商品編集

> ワイヤーフレーム: `wireframes/a-04-product-create.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-04-product-create.html)
> PC専用。A-05（商品編集）は同じレイアウトで既存データが入力済みの状態。

---

## 1. 画面概要

| 項目 | A-04 | A-05 |
|------|------|------|
| 画面名 | 商品登録 | 商品編集 |
| URL | `/admin/products/new` | `/admin/products/[id]/edit` |
| 認証 | 必要（管理者） | 必要（管理者） |

---

## 2. セクション構成

| セクション | 内容 |
|------------|------|
| 基本情報 | 商品名・カテゴリ・おすすめ度・説明・公開ステータス |
| 商品画像 | 複数画像アップロード・並び替え。オプション選択との連動 |
| オプション設定 | グループ（画像有無・価格タイプ）＋選択肢（画像・追加価格・ヘルプ・バッジ・デフォルト） |
| 価格テーブル | 数量帯ごとの単価＋納期（営業日）。注記テキスト |
| 商品詳細情報 | 仕様テーブル（素材・サイズ展開等）＋注記テキスト |
| 送料 | 商品ごとの送料設定 |
| サンプル制作 | サンプル受付有無＋単価＋概算納期 |
| 入稿設定 | 入稿仕様（リッチテキスト）＋スロット定義＋テンプレートファイル |
| 関連商品 | この商品に関連する商品の選択 |
| 商品FAQ | この商品固有のFAQ |

---

## 3. セクション詳細

### 3.1 基本情報

| 項目 | 必須/任意 | 入力形式 | 備考 |
|------|----------|----------|------|
| 商品名 | 必須 | テキスト | — |
| カテゴリ | 必須 | プルダウン | A-11で管理するカテゴリから選択 |
| おすすめ度 | 任意 | 5段階（★） | C-02商品一覧の「おすすめ順」ソートに使用。デフォルト0 |
| 商品説明 | 必須 | テキストエリア | C-03商品詳細に表示 |
| 公開ステータス | — | プルダウン | 「非公開（下書き）」「公開」。デフォルト非公開 |

### 3.2 商品画像

| 項目 | 詳細 |
|------|------|
| アップロード | 複数画像アップロード可能。ドラッグで並び替え |
| メイン画像 | 1枚目がメイン画像として一覧・カートに表示 |
| ホバー画像 | 2枚目がホバー時の切替画像（C-01カテゴリ、C-02一覧、C-03関連商品） |

### 3.3 オプション設定

C-03画面設計書 3.3.3節で定義されたオプション仕様に基づく管理UI。オプション項目を動的に追加し、各選択肢を個別に管理する。

#### オプショングループの設定項目

| 項目 | 必須/任意 | 詳細 |
|------|----------|------|
| グループ名 | 必須 | 「サイズ」「素材・仕上げ」等 |
| グループヘルプテキスト | 任意 | C-03でグループ名横の「?」アイコンクリック時に表示。未設定の場合アイコン非表示 |
| 画像の有無 | 必須 | 選択肢に画像を表示するか（はい/いいえ） |
| 価格タイプ | 必須 | なし / 単価系（数量に比例して加算）/ 固定費（数量に関わらず一律加算） |

#### 各選択肢の設定項目（グループ内）

| 項目 | 必須/任意 | 詳細 |
|------|----------|------|
| 選択肢名 | 必須 | 「44mm」「マット」「安全ピン」等 |
| 選択肢の画像 | 画像ありの場合のみ | 選択肢ごとのサムネイル画像 |
| 追加価格 | 価格タイプがある場合のみ | 単価系: 「+10円」等（単価に加算）、固定費: 「+3,000円」等（一律加算） |
| ヘルプテキスト | 任意 | C-03で選択肢横の「?」アイコンクリック時に表示 |
| バッジ | 任意 | 「人気」等のラベル表示 |
| デフォルト選択 | 必須（グループ内で1つ） | ページ表示時にこの選択肢が初期選択される |

#### オプションパターンの例（C-03 3.3.3節より）

| パターン | 画像 | 価格タイプ | 例 |
|----------|------|-----------|-----|
| A | なし | なし | サイズ、納品先 |
| B | あり | なし | 素材・仕上げ |
| C | あり | 単価系（+N円） | 付属パーツ |
| D | なし | 単価系（+N円） | 包装、裏面デザイン、納期 |
| E | なし | 固定費（+N円） | 色校正（あり/なし） |
| F | なし | 固定費（+N円） | 入稿サポート（あり/なし） |

### 3.4 価格テーブル

| 列 | 詳細 |
|----|------|
| 数量（個） | 数量帯の下限値。1行目が最小ロット |
| 単価（税込） | その数量帯の単価。最小ロット時の単価がC-04で定価として表示 |
| 納期（営業日） | その数量帯の納期目安。入金確認完了後から起算 |
| 行の追加/削除 | 動的に増減可能 |
| 注記テキスト | 価格テーブル下の注意事項。テキスト入力。C-03の価格テーブルタブに表示 |

### 3.5 商品詳細情報

C-03のタブ「商品詳細」に表示される仕様テーブル。

| 項目 | 詳細 |
|------|------|
| 仕様テーブル | ラベル＋値の2列テーブル。行を動的追加/削除。例: 素材=アクリル、サイズ展開=44mm/56mm/76mm |
| 注記テキスト | テーブル下の注意事項。テキスト入力 |

### 3.6 送料

| 項目 | 詳細 |
|------|------|
| 送料（税込） | 商品ごとの送料。サイト全体の送料無料閾値を超えた場合は自動無料 |

### 3.7 サンプル制作

| 項目 | 詳細 |
|------|------|
| 受付有無 | はい/いいえ。「いいえ」の場合C-03にサンプルボタン非表示 |
| サンプル単価 | サンプル1個あたりの価格（税込） |
| 概算納期 | サンプル制作の概算納期（営業日）。C-03のサンプル確認モーダルに表示 |

### 3.8 入稿設定

#### 入稿仕様

| 項目 | 詳細 |
|------|------|
| 入稿仕様本文 | リッチテキスト（WYSIWYG）で自由に記述。印刷方法、対応形式、カラーモード、解像度、注意事項等を商品に応じて記載。C-03のタブ「入稿仕様」に表示 |

#### 入稿スロット

| 項目 | 詳細 |
|------|------|
| 動的追加 | 「スロットを追加」で増やせる |
| ラベル | スロット名（例: 表面デザイン、裏面デザイン、台紙デザイン） |
| 必須/任意 | プルダウン |
| 表示条件 | 「常に表示」or「特定オプション選択時」 |
| 表示条件の設定 | 「特定オプション選択時」を選んだ場合、対象のオプショングループ（プルダウン）＋対象の選択肢値（プルダウン）を指定。例: 「付属」=「マグネット」の時のみ表示 |
| テンプレートファイル | スロットごとにテンプレートファイルをアップロード。C-03・C-05からダウンロード可能 |

#### 入稿スロットの設定例

| スロット | ラベル | 必須/任意 | 表示条件 |
|---------|--------|----------|---------|
| 1 | 表面デザイン | 必須 | 常に表示 |
| 2 | 裏面デザイン | 任意 | 常に表示 |
| 3 | 台紙デザイン | 必須 | 付属 = マグネット の場合のみ |

### 3.9 関連商品

| 項目 | 詳細 |
|------|------|
| 選択方法 | 登録済み商品から検索して選択。最大4件 |
| 表示先 | C-03商品詳細ページの「関連商品」セクション |

### 3.10 商品FAQ

| 項目 | 詳細 |
|------|------|
| 動的追加 | 「FAQを追加」で増やせる |
| 各FAQ | 質問（テキスト）＋ 回答（テキストエリア） |
| 削除 | ×ボタンで削除 |
| 表示先 | C-03商品詳細ページのFAQセクション。グローバルFAQ（ADM-008）とは別管理 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 「商品を登録する」クリック | バリデーション実行。成功 → 保存してA-03に遷移 |
| 「キャンセル」クリック | 変更を破棄してA-03に遷移 |
| 「商品一覧に戻る」クリック | A-03に遷移（変更がある場合は確認ダイアログ） |
| 画像ドラッグ | 画像の並び順を変更 |
| 各「追加」ボタン | オプション/数量帯/固定費/スロット/FAQを動的追加 |
| 各「×」ボタン | 該当項目を削除（確認ダイアログなし） |

---

## 5. バリデーション

| 条件 | 表示 |
|------|------|
| 商品名が未入力 | 「商品名を入力してください」 |
| カテゴリが未選択 | 「カテゴリを選択してください」 |
| 商品説明が未入力 | 「商品説明を入力してください」 |
| 価格テーブルが0行 | 「価格テーブルを1行以上設定してください」 |
| 入稿スロットが0件 | 「入稿スロットを1件以上設定してください」 |
