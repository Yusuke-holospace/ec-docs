# 画面設計書：共通フッター

全購入者向けページ（C-01〜C-20）で共通のフッター構成。
C-08（ログイン）/ C-09（会員登録）/ C-21（共有ステータス）はミニマルフッター（コピーライトのみ）のため対象外。

---

## 1. PC版フッター

### 1.1 フッター本体（3カラム構成）

| カラム | 見出し | リンク | 遷移先 |
|--------|--------|--------|--------|
| 商品カテゴリ | 商品カテゴリ | 缶バッジ | C-02（カテゴリフィルタ付き） |
| | | アクリルスタンド | C-02 |
| | | キーホルダー | C-02 |
| | | その他グッズ | C-02 |
| サポート | サポート | FAQ | C-15 |
| | | ご利用ガイド | C-16 |
| | | お問い合わせ | C-14 |
| 法的情報 | 法的情報 | 利用規約 | C-17 |
| | | プライバシーポリシー | C-18 |
| | | 特定商取引法に基づく表記 | C-19 |
| | | コーポレートサイト | 外部リンク |

### 1.2 コピーライト

| 要素 | 詳細 |
|------|------|
| テキスト | &copy; 2026 Company Name. All rights reserved. |
| 配置 | 中央揃え。上部に区切り線（border-top: 1px solid #555） |

### 1.3 スタイル

| 項目 | 値 |
|------|-----|
| 背景色 | #333 |
| テキスト色 | #fff（見出し）/ #ccc（リンク） |
| コピーライト色 | #999 |
| padding | 40px 24px |
| 最大幅 | 1200px（中央寄せ） |
| カラムgap | 32px |

---

## 2. SP版フッター

### 2.1 構成

PC版と同じ3セクション（商品カテゴリ / サポート / 法的情報）を**縦積み**で表示。

### 2.2 PC版との差分

| 項目 | PC版 | SP版 |
|------|------|------|
| レイアウト | 3カラム横並び | 縦積み |
| padding | 40px 24px | 24px 16px |
| 見出しフォントサイズ | 14px | 13px |
| リンクフォントサイズ | 13px | 12px |
| コピーライトフォントサイズ | 12px | 11px |
| 法的情報の表記 | 「特定商取引法に基づく表記」 | 「特商法表記」（短縮） |
| 商品カテゴリリンク数 | 4件 | 3件（その他グッズを省略） |

> SP版のリンク数・表記の差異は画面幅の制約による。統一が望ましい場合は要検討。

---

## 3. パンくずリスト

フッター直上にパンくずリストを設置（PC版のみ）。ヘッダー直下のパンくずと同じ内容。

| 項目 | 詳細 |
|------|------|
| 表示 | PC版のみ。SP版は非表示 |
| 配置 | フッターの直上（margin/gap なし） |
| スタイル | font-size: 13px、color: #999、リンク色: #666 |
| 非表示ページ | C-01（トップページ） |

---

## 4. ページ別の差分

| ページ | 差分 |
|--------|------|
| C-01 トップページ | フッター直上のパンくずなし |
| C-08 ログイン / C-09 会員登録 | フッターなし |
| C-21 共有ステータスページ | ミニマルフッター（コピーライトのみ、3カラムなし） |
| その他全ページ | 共通フッター（本ドキュメントの通り） |
