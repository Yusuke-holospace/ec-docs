---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | A-12 |
| 画面名 | ヒーローバナー管理 |
| URL | `/admin/banners` |
| 認証 | 必要（管理者） |

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル＋新規追加ボタン | 右上に「新規追加」ボタン |
| 説明テキスト | 「C-01 トップページのカルーセルに表示されるバナーを管理します。」 |
| バナーリスト | ドラッグ並び替え可能なカード形式リスト |
| 編集フォーム | バナー追加・編集フォーム |

---

## 3. セクション詳細

### 3.1 バナーリスト

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| ドラッグハンドル | 並び順変更用のグリップアイコン | — |
| 並び順番号 | 現在の表示順 | 管理者 |
| バナー画像プレビュー | 200x80pxのプレビュー表示 | 管理者 |
| alt テキスト | バナーの代替テキスト（太字表示） | 管理者 |
| リンクURL | クリック時の遷移先URL | 管理者 |
| 操作 | 「編集」「削除」 | — |

### 3.2 編集フォーム

| フィールド | 詳細 |
|------------|------|
| バナー画像 | 画像アップロード（推奨: 1200x400px、JPG/PNG） |
| リンクURL | テキスト入力。バナークリック時の遷移先 |
| alt テキスト | テキスト入力。画像の代替テキスト |
| ボタン | 「保存」「キャンセル」 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 「新規追加」クリック | 空の編集フォームを表示 |
| 「編集」クリック | 該当バナーの編集フォームを表示 |
| 「削除」クリック | 確認ダイアログ表示。確認後に削除 |
| ドラッグ&ドロップ | 表示順を変更。変更は自動保存。C-01カルーセルの表示順に即時反映 |
| 「保存」クリック | バナー情報を保存 |
| 「キャンセル」クリック | 変更を破棄してフォームを閉じる |
