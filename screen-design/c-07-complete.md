# 画面設計書：C-07 注文完了ページ

> ワイヤーフレーム PC: `wireframes/c-07-complete.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/c-07-complete.html)
> ワイヤーフレーム SP: `wireframes/c-07-complete-sp.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/c-07-complete-sp.html)

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-07 |
| 画面名 | 注文完了ページ |
| URL | `/cart/complete` |
| 認証 | 必要 |
| 対応デバイス | PC・スマートフォン（レスポンシブ） |

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | PC: C-03と同じ / SP: C-03 SP版と同じ |
| パンくずリスト | トップ > 注文完了（PC版のみ） |
| ステップインジケーター | カート → 入稿 → 確認 → **完了** |
| 完了メッセージ | アイコン＋タイトル＋説明文 |
| 注文番号 | 注文番号を表示 |
| 今後の流れ | 入稿確認→決済案内→製造→発送の4ステップ |
| アクションボタン | 「注文履歴を見る」「商品一覧に戻る」 |
| フッター（共通） | PC: C-03と同じ / SP: C-03 SP版と同じ |

---

## 3. セクション詳細

### 3.1 完了メッセージ

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| アイコン | プレースホルダー（緑の丸背景） | — |
| タイトル | 「ご注文を受け付けました」 | 固定テキスト |
| 説明文 | 「ご注文ありがとうございます。注文内容の確認メールをお送りしました。」 | 固定テキスト |

### 3.2 注文番号

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| ラベル | 「注文番号」 | 固定テキスト |
| 番号 | 自動採番（例: ORD-20260414-0023） | システム |

### 3.3 今後の流れ

| ステップ | 内容 | 設定元 |
|----------|------|--------|
| 1 | 入稿データを確認いたします（数営業日） | 固定テキスト |
| 2 | 確認完了後、決済のご案内をメールでお送りします | 固定テキスト |
| 3 | お支払い確認後、製造を開始します | 固定テキスト |
| 4 | 製造完了後、発送いたします | 固定テキスト |

### 3.4 共有URLコピー

| 要素 | 詳細 |
|------|------|
| 「共有URLをコピー」ボタン | 共有ステータスページ（C-21）のURLをクリップボードにコピー |
| 用途 | 注文完了直後に社内の他の担当者へステータス確認ページを共有 |

### 3.5 アクションボタン

| 要素 | 詳細 |
|------|------|
| 注文履歴を見る | C-12（注文履歴一覧ページ）へ遷移 |
| 商品一覧に戻る | C-02（商品一覧ページ）へ遷移 |

---

## 4. 補足

- 注文確定時にカートは空になる（ヘッダーのカートバッジは0に）
- 確認メール（NTF-002）が購入者に、新規注文通知（NTF-008）が管理者に送信される
- この画面は注文確定後に1回だけ表示される。ブラウザバックで戻れないようにする
