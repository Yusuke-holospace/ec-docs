# 画面設計書：共通ヘッダー

全購入者向けページ（C-01〜C-20）で共通のヘッダー構成。
C-08（ログイン）/ C-09（会員登録）/ C-21（共有ステータス）はミニマルヘッダー（ロゴのみ）のため対象外。

---

## 1. PC版ヘッダー（3段構成）

### 1.1 告知バー（最上部）

| 要素 | 詳細 |
|------|------|
| 背景色 | #333（黒） |
| テキスト | 「デザインデータをそのまま送るだけ。入稿データの編集はスタッフが無料でサポートします。」 |
| アイコン | `pencil-line`（Lucide） |
| 配置 | 中央揃え |
| 用途 | 入稿サポートの訴求。全ページで常時表示 |

> 将来的にキャンペーン告知等に差し替え可能。管理画面からの編集は初期スコープ外。

### 1.2 ヘッダー1段目

| 要素 | 詳細 |
|------|------|
| **左: ロゴ** | ロゴ画像プレースホルダー（120×36px）。クリックでC-01（トップページ）へ遷移 |
| **中央: 検索窓** | 角丸入力欄（幅: 35vw、センター固定）+ 検索アイコン（`search`）を入力欄内右側に配置 |
| **右: ユーティリティリンク** | アイコン＋テキストの縦積み。以下5つ |

**ユーティリティリンク:**

| リンク | アイコン（Lucide） | 遷移先 |
|--------|-------------------|--------|
| ご利用ガイド | `book-open` | C-16 |
| FAQ | `help-circle` | C-15 |
| 入稿について | `file-text` | C-16（入稿セクション） |
| ログイン | `log-in` | C-08 |
| カート | `shopping-cart` | C-04 |

> ログイン済みの場合: 「ログイン」→「マイページ」（アイコン: `user`、遷移先: C-10）に切替
> カートアイコンにはバッジ（赤丸＋商品数）を表示

### 1.3 ヘッダー2段目（カテゴリナビゲーション）

| 要素 | 詳細 |
|------|------|
| 配置 | 横並び。ヘッダー1段目の直下 |
| スタイル | 太字テキスト。ホバーで下線（border-bottom: 3px） |
| ドロップダウン | カテゴリにホバーでメガメニューを展開（画像+商品名+価格のグリッド） |

**カテゴリ一覧:**
缶バッジ / アクリルスタンド / キーホルダー / ストラップ / トレーディングカード / タオル・布製品 / その他グッズ

> ドロップダウンは「缶バッジ」のみ実装済み（サブ商品3つ）。他カテゴリも同様の構造で展開。
> クリックでC-02（商品一覧）にカテゴリフィルタ付きで遷移。

## 2. SP版ヘッダー（2段構成）

### 2.1 ヘッダー1段目

| 要素 | 詳細 |
|------|------|
| 左: ロゴ | ロゴテキスト（18px）。クリックでC-01へ |
| 右: ナビアイコン | アイコン+テキストの縦積み。3つ |

**ナビアイコン:**

| リンク | アイコン | 遷移先 |
|--------|---------|--------|
| ログイン | nav-icon プレースホルダー | C-08 |
| カート | nav-icon + バッジ | C-04 |
| メニュー | nav-icon プレースホルダー | ハンバーガーメニュー展開 |

> ログイン済みの場合: 「ログイン」→「マイページ」に切替
> タップ領域: 全要素 min 44×44px

### 2.2 ヘッダー2段目（検索バー）

| 要素 | 詳細 |
|------|------|
| 検索入力欄 | 幅100%（左右padding: 16px）|
| 検索ボタン | 「検索」テキストボタン |

> カテゴリナビはSP版では非表示（メニューから遷移）
> 告知バーはSP版では非表示（画面スペース節約のため。要検討）

## 3. ページ別の差分

| ページ | 差分 |
|--------|------|
| C-01 トップページ | パンくずなし |
| C-08 ログイン / C-09 会員登録 | ミニマルヘッダー（ロゴのみ。カテゴリナビ・ユーティリティリンクなし） |
| C-21 共有ステータスページ | ミニマルヘッダー（ロゴ + 「ECサイトへ」リンクのみ） |
| その他全ページ | 共通ヘッダー（本ドキュメントの通り） |

---

## 4. アイコンライブラリ

| 項目 | 詳細 |
|------|------|
| ライブラリ | **Lucide** |
| 理由 | Next.js（React）との相性が良い。`lucide-react`パッケージで提供。Tree-shaking対応で軽量 |
| CDN（ワイヤーフレーム用） | `https://unpkg.com/lucide@latest` |
| 初期化 | `lucide.createIcons()` をページ末尾で呼び出し |
