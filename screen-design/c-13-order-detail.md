## 2. セクション詳細

### 2.1 ステータス

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| ステータスバッジ | 現在のステータスを色付きバッジで表示 | サーバー |
| 注文番号 | ORD-XXXXXXXX-XXXX | サーバー |
| 注文日 | YYYY/MM/DD | サーバー |

### 2.2 ステータスタイムライン

注文のステータス遷移を縦のタイムラインで表示。完了済みステップは緑、現在のステップは黒、未到達はグレー。各ステップに日時を表示（完了済みのみ）。

| ステップ | 表示 |
|----------|------|
| 注文受付 | 入稿受付 or サンプル受付 |
| 入稿確認中 | 通常注文のみ |
| 承認済み | 通常注文のみ |
| 決済待ち | — |
| 入金確認 | — |
| 製造中 | — |
| 発送済み | — |
| 完了 | — |

### 2.3 注文商品一覧

C-06（注文確認ページ）と同じ形式。商品画像・種別バッジ・商品名・オプション・入稿状態・金額。

### 2.4 お支払い

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 支払い方法 | クレジットカード / 銀行振込 / 請求書払い | サーバー |
| 商品合計 | 全商品の小計合計 | サーバー |
| 送料 | 無料 or 金額 | サーバー |
| 合計（税込） | 商品合計＋送料 | サーバー |

### 2.5 配送先

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 住所 | 注文時の配送先（会社名・担当者名・住所・電話番号） | サーバー |

### 2.6 操作ボタン

ステータスに応じて表示されるボタンが変わる。

| ステータス | 表示ボタン | 遷移先 |
|------------|-----------|--------|
| 差し戻し | 再入稿する | C-05（入稿ページ・再入稿モード） |
| 完了 | リピート注文する | カートに追加 → C-04（カートページ） |
| その他 | ボタンなし | — |
