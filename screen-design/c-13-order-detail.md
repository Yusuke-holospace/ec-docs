## 2. セクション詳細

### 2.1 ステータス

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| ステータスバッジ | 現在のステータスを色付きバッジで表示 | サーバー |
| 注文番号 | ORD-XXXXXXXX-XXXX | サーバー |
| 注文日 | YYYY/MM/DD | サーバー |

### 2.2 差し戻しアラート（条件表示）

差し戻し状態の商品がある場合のみ、ステータスバッジの下にアラートバナーを表示。

| 要素 | 詳細 |
|------|------|
| 背景色 | 赤（#fde8e8） |
| テキスト | 「入稿データに不備があります。該当商品の「再入稿する」ボタンから修正データをアップロードしてください。」 |

> アラートにはボタンを置かない。再入稿ボタンは商品一覧の該当商品行に表示する。

### 2.3 ステータスタイムライン

注文のステータス遷移を縦のタイムラインで表示。完了済みステップは緑、現在のステップは黒、未到達はグレー。各ステップに日時を表示（完了済みのみ）。

| ステップ | 表示 |
|----------|------|
| 注文受付 | 入稿受付 or サンプル受付 |
| 入稿確認中 | 通常注文のみ |
| 承認済み | 通常注文のみ |
| 決済待ち | — |
| 入金確認 | — |
| 製造中 | — |
| 発送済み | — |
| 完了 | — |

### 2.4 注文商品一覧

C-06（注文確認ページ）と同じ形式。商品画像・種別バッジ・商品名・オプション・入稿状態・金額。

差し戻された商品には以下を追加表示:
- 入稿状態を「差し戻し」（赤文字）で表示
- 「再入稿する」ボタン → C-05（入稿ページ・再入稿モード）にその商品の入稿画面として遷移
- 商品行の背景色を薄赤（#fff8f8）に変更

### 2.5 お支払い

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 支払い方法 | クレジットカード / 銀行振込 / 請求書払い | サーバー |
| 商品合計 | 全商品の小計合計 | サーバー |
| 送料 | 無料 or 金額 | サーバー |
| 合計（税込） | 商品合計＋送料 | サーバー |

### 2.6 配送先

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 住所 | 注文時の配送先（会社名・担当者名・住所・電話番号） | サーバー |

### 2.7 担当スタッフ

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 担当者カード | アイコン + 担当者名 + 「ご不明点がございましたらお気軽にお問い合わせください」 | サーバー（ADM-014で設定） |

> 担当者未割り当ての場合は「担当者が確定次第、こちらに表示されます」と表示。

### 2.8 仕上がりイメージ（条件表示）

管理者が仕上がりイメージをアップロード済みの場合のみ表示。注文商品一覧の直後に配置。

| 要素 | 詳細 |
|------|------|
| サムネイル一覧 | 3列グリッドで画像を表示 |
| 説明テキスト | 「販促用素材としてご利用いただけます（背景透過PNG含む）」 |
| 「一括ダウンロード（ZIP）」ボタン | 全画像をZIPにまとめてダウンロード |

### 2.9 共有URLコピー

| 要素 | 詳細 |
|------|------|
| 配置 | ページタイトル「注文詳細」の右側にシェアアイコンとして配置 |
| 動作 | 共有ステータスページ（C-21）のURLをクリップボードにコピー |

> 独立セクションではなく、ページヘッダー内のシェアボタンとして表示。

### 2.10 操作ボタン

ステータスに応じて表示されるボタンが変わる。

| ステータス | 表示ボタン | 遷移先 |
|------------|-----------|--------|
| 完了 | リピート注文する | カートに追加 → C-04（カートページ） |
| その他 | ボタンなし | — |

> 再入稿ボタンは2.4（注文商品一覧）の該当商品行に表示。ページ下部の操作ボタンには含めない。
