# 画面設計書：C-03 商品詳細ページ

> ワイヤーフレーム: `wireframes/c-03-product-detail.html`

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-03 |
| 画面名 | 商品詳細ページ |
| URL | `/products/[slug]` |
| 認証 | 不要（閲覧のみ。カート追加・サンプル制作はログイン必要） |
| 対応デバイス | PC・スマートフォン（レスポンシブ） |

---

## 2. 画面構成

上から順に以下のセクションで構成される。

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | 2段構成。1段目: ロゴ・検索・ユーティリティリンク・カート。2段目: カテゴリナビ（ドロップダウン） |
| パンくずリスト | トップ > カテゴリ > 商品名 |
| 商品上部（左） | 商品画像ギャラリー |
| 商品上部（右） | 商品情報・オプション選択・数量入力・価格表示・アクションボタン |
| タブセクション | 価格テーブル / 商品詳細 / 入稿仕様（タブ切替） |
| 関連商品 | 関連商品カード4件 |
| 商品FAQ | 個別商品FAQ ＋ グローバル商品FAQをマージ表示（アコーディオン） |
| パンくずリスト（下部） | ヘッダー下と同じ内容 |
| フッター（共通） | サポート・法的情報・コーポレートサイトリンク |

---

## 3. セクション詳細

### 3.1 ヘッダー（共通コンポーネント）

#### 1段目
| 要素 | 詳細 |
|------|------|
| ロゴ | クリックでトップページ（C-01）へ |
| 検索バー | キーワード入力 → 商品一覧ページ（C-02）へ遷移 |
| ご利用ガイド | C-17へ |
| FAQ | C-16へ |
| 入稿について | C-17（ご利用ガイド内の入稿セクション）へ |
| ログイン / マイページ | 未ログイン時:「ログイン」を表示（C-08へ）。ログイン済み:「マイページ」を表示（C-10へ）。表示が切り替わる |
| カート | C-04へ。カート内商品数をバッジ表示 |

#### 2段目（カテゴリナビ）
| 要素 | 詳細 |
|------|------|
| カテゴリ名 | 横並びで表示 |
| ホバー時 | ドロップダウンでカテゴリ内の商品を表示（画像＋商品名＋説明の3列グリッド） |
| カテゴリ名クリック | C-02（該当カテゴリの商品一覧）へ遷移 |
| ドロップダウン内の商品クリック | 該当商品のC-03（商品詳細ページ）へ遷移 |

### 3.2 商品画像ギャラリー（左カラム）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| メイン画像 | 選択中のサムネイルを大きく表示。オプション選択に連動して切り替わる | 商品別 |
| サムネイル一覧 | 横並び。正面・裏面・使用例・印刷範囲 等 | 商品別 |
| サムネイルクリック | メイン画像が切り替わる。選択中は枠線で強調 | — |

### 3.3 商品情報（右カラム）

3つのエリアに分かれる。オプションが多い場合でも価格・ボタンが常にFV内に表示されるよう、価格エリアはsticky。

#### 3.3.1 商品ヘッダー（固定）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| カテゴリ名 | 商品が属するカテゴリ。クリックでC-02へ | 商品別 |
| 商品名 | h1。商品の正式名称 | 商品別 |
| 商品説明 | 商品の概要テキスト | 商品別 |

#### 3.3.2 オプション選択エリア（スクロール可能）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| テキストオプション | サイズ等。ボタン形式で選択。選択中は枠線＋背景で強調 | 商品別 |
| 画像付きオプション | 素材・仕上げ・付属パーツ等。4列グリッドのカード形式 | 商品別 |
| オプションカード | 画像（任意）＋名前＋追加価格（+0円、+10円等）を表示 | 商品別 |
| バッジ | 「人気」等のラベル表示（任意） | 商品別 |
| 数量入力 | 数値入力フィールド。最小ロットのバリデーション付き | — |
| 最小ロット表示 | 数量入力の下に「最小ロット: 100個」等を表示 | 商品別 |

#### 3.3.3 価格・アクションエリア（sticky固定）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 単価 | オプション・数量に連動してリアルタイム更新 | 商品別（価格テーブルから算出） |
| 小計 | 単価 × 数量。税込表示 | 自動算出 |
| 送料 | N円以上で無料の旨を表示 | 共通（サイト設定） |
| 納期目安 | 数量に応じた納期を表示 | 商品別 |
| カートに追加ボタン | プライマリボタン。未ログイン時はC-08へ遷移後、商品ページに戻りカートに追加 | 固定テキスト |
| サンプル制作ボタン | セカンダリボタン。サンプル制作としてカートに追加 | 固定テキスト |
| サブリンク | 見積書DL（PDF）、入稿テンプレートDL、この商品について問い合わせる（C-14へ） | 固定テキスト |

### 3.4 タブセクション

タブの切替で3つのコンテンツを表示。デフォルトは「価格テーブル」が選択状態。

#### タブ1: 価格テーブル

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| テーブル | 数量範囲別の単価・合計一覧（税込）。「100〜199」「200〜299」のように範囲表記。最小ロット以上なら任意の数量で注文可能 | 商品別 |
| 注記 | テーブル下に注意事項を表示 | 商品別 |

#### タブ2: 商品詳細

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 詳細テーブル | 素材、サイズ展開、仕上げ、最小ロット、納期目安 等 | 商品別 |
| 表示形式 | ラベル（左）＋値（右）の2列テーブル | — |
| 注記 | テーブル下に注意事項を表示 | 商品別 |

#### タブ3: 入稿仕様

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 入稿仕様本文 | リッチテキスト（WYSIWYG）で自由に記述。見出し・箇条書き・太字・画像等を使用可能。印刷方法、対応形式、カラーモード、解像度、注意事項等を商品に応じて記載 | 商品別 |
| テンプレートDL | 入稿テンプレートのダウンロードリンク（選択中のオプションに対応） | 商品別 |

### 3.5 関連商品

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 表示件数 | 4件（PC）、2件（スマホ） | 固定 |
| カード構成 | 商品画像 + 商品名 + 最低単価 | 商品別（管理者が関連商品を設定） |
| クリック | 該当商品のC-03へ遷移 | — |

### 3.6 商品FAQ

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 個別商品FAQ | この商品に紐づくFAQ | 商品別（ADM-001） |
| グローバル商品FAQ | 全商品共通のFAQ | 共通（ADM-008） |
| 表示 | 上記をマージして表示 | — |
| 表示形式 | アコーディオン（クリックで開閉） | — |
| デフォルト | 最初の1件は開いた状態 | — |

### 3.7 パンくずリスト（下部）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| パンくず | ヘッダー下と同じ内容。トップ > カテゴリ > 商品名 | 自動生成 |

### 3.8 フッター（共通コンポーネント）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 商品カテゴリ | カテゴリ一覧へのリンク。クリックでC-02へ | 共通（カテゴリマスタ） |
| サポート | FAQ（C-16）、ご利用ガイド（C-17）、お問い合わせ（C-15）へのリンク | 固定テキスト |
| 法的情報 | 利用規約（C-18）、プライバシーポリシー（C-19）、特商法表記（C-20）へのリンク | 固定テキスト |
| コーポレートサイト | 既存コーポレートサイトへの外部リンク | 共通（サイト設定） |
| コピーライト | &copy; 2026 Company Name. All rights reserved. | 固定テキスト |

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| オプション選択 | 商品画像が連動して切り替わる。価格・納期がリアルタイム更新 |
| 数量入力 | 価格・納期・価格テーブルのハイライトがリアルタイム更新 |
| カートに追加 | 未ログイン → C-08 → ログイン後C-03に戻りカートに追加。ログイン済み → カート追加完了モーダルを表示（下記参照） |
| サンプル制作をカートに追加 | 上記と同様。カートにサンプル種別で追加 |

**カート追加完了モーダル:**

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| タイトル | 「商品をカートに入れました」 | 固定テキスト |
| カート情報 | 合計点数、合計金額（税込） | 自動算出 |
| 「買い物を続ける」ボタン | モーダルを閉じてC-03に留まる | 固定テキスト |
| 「カートを見る」ボタン | C-04（カートページ）へ遷移 | 固定テキスト |
| 閉じるボタン（×） | モーダルを閉じる | — |
| 見積書DL | 現在のオプション・数量で見積書PDFを生成・ダウンロード |
| テンプレートDL | 選択中のオプションに対応するテンプレートファイルをダウンロード |
| 問い合わせ | C-14へ遷移（商品情報が自動セット） |
| タブ切替 | 価格テーブル / 商品詳細 / 入稿仕様 を切替表示 |
| FAQアコーディオン | クリックで開閉 |
| 関連商品クリック | 該当商品のC-03へ遷移 |
| カテゴリホバー | ドロップダウンでサブカテゴリ表示 |

---

## 5. バリデーション・エラー

| 条件 | 表示 |
|------|------|
| 最小ロット未満の数量入力 | 数量入力の下に赤字で「最小ロットはXX個です」と表示。カートに追加ボタンを非活性化 |
| オプション未選択 | 必須オプションが未選択の場合、カートに追加ボタンを非活性化 |

---

## 6. データ要件

この画面の表示に必要なデータ。設定元が「商品別」のものは管理者が商品ごとに設定、「共通」は全商品で同じ内容が表示される。

| データ | 内容 | 設定元 |
|--------|------|--------|
| 商品名 | 商品の正式名称 | 商品別 |
| 商品説明文 | 商品の概要テキスト | 商品別 |
| カテゴリ | 商品が属するカテゴリ | 商品別 |
| 商品画像 | 複数枚。オプションとの紐づけ情報 | 商品別 |
| オプション定義 | オプショングループ（サイズ、仕上げ等）ごとの選択肢。各選択肢に画像（任意）・追加価格・バッジ | 商品別 |
| 価格テーブル | 数量別の単価・合計 | 商品別 |
| 価格テーブル注記 | 価格テーブル下の注意事項テキスト | 商品別 |
| 納期テーブル | 数量別の納期目安 | 商品別 |
| 最小ロット | 通常注文・サンプル制作それぞれの最小数量 | 商品別 |
| 商品詳細情報 | 素材、サイズ展開、仕上げ等の仕様テーブル | 商品別 |
| 商品詳細注記 | 商品詳細下の注意事項テキスト | 商品別 |
| 入稿仕様 | 印刷方法、対応形式、カラーモード、解像度等 | 商品別 |
| 入稿仕様注記 | 入稿仕様下の注意事項テキスト | 商品別 |
| 入稿テンプレート | オプション別のテンプレートファイル | 商品別 |
| 関連商品 | 関連商品のリスト（商品ID、画像、名前、最低単価） | 商品別 |
| 個別商品FAQ | この商品に紐づくFAQ（質問・回答のリスト） | 商品別（ADM-001） |
| グローバル商品FAQ | 全商品共通のFAQ（質問・回答のリスト） | 共通（ADM-008） |
| 送料無料の閾値 | N円以上で送料無料 | 共通（サイト設定） |
