# 画面設計書：C-03 商品詳細ページ

> ワイヤーフレーム PC: `wireframes/c-03-product-detail.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/c-03-product-detail.html)
> ワイヤーフレーム SP: `wireframes/c-03-product-detail-sp.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/c-03-product-detail-sp.html)

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-03 |
| 画面名 | 商品詳細ページ |
| URL | `/products/[slug]` |
| 認証 | 不要（閲覧・カート追加ともにログイン不要。入稿・注文確定時にログイン必要） |
| 対応デバイス | PC・スマートフォン（レスポンシブ。1つのHTMLで対応） |

---

## 2. 画面構成

### PC版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | 2段構成。1段目: ロゴ・検索・ユーティリティリンク・カート。2段目: カテゴリナビ（ドロップダウン） |
| パンくずリスト | トップ > カテゴリ > 商品名 |
| 商品上部（左） | 商品名・説明・オプション選択・数量入力 |
| 商品上部（右・sticky） | 画像ギャラリー・価格表示（内訳展開）・納期目安・カートに追加ボタン・サブリンク |
| タブセクション | 価格テーブル / 商品詳細 / 入稿仕様（タブ切替） |
| 関連商品 | 関連商品カード4件（グリッド表示） |
| 商品FAQ | 個別商品FAQ ＋ グローバル商品FAQをマージ表示（アコーディオン） |
| パンくずリスト（下部） | ヘッダー下と同じ内容。フッターとの隙間なし |
| フッター（共通） | サポート・法的情報・コーポレートサイトリンク |

### SP版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | 1段目: ロゴ・ログイン・カート・メニュー（アイコン+テキスト）。2段目: 検索バー+検索ボタン |
| 画像ギャラリー | 全幅画像。スワイプで切替、ピンチズーム対応。ドットインジケーター（サムネイルなし） |
| 商品情報 | 商品名・価格・納期・内訳展開・商品説明（「もっと見る」で展開） |
| オプション選択 | PC版と同様（3列グリッド） |
| サブリンク | 見積書DL・テンプレートDL・問い合わせ（3列グリッド） |
| アコーディオンセクション | 価格テーブル / 商品詳細 / 入稿仕様（PCのタブをアコーディオンに変更） |
| 関連商品 | 横スクロールで4件表示 |
| 商品FAQ | PC版と同じアコーディオン |
| フッター（共通） | PC版と同じ |
| 下部CTAバー（sticky） | 価格＋1個あたり単価＋カートに追加ボタン＋サンプル制作ボタン。画面最下部に常時固定 |

**SP版でのPC版との主な差分：**
- パンくずリスト: 非表示
- タブ → アコーディオン
- 画像サムネイル → ドットインジケーター
- 商品説明: 2行で折りたたみ（「もっと見る」で展開）
- 関連商品: グリッド → 横スクロール
- 価格・CTAボタン: 下部固定バー
- カートモーダル: 中央表示 → ボトムシート

## 3. セクション詳細

### 3.1 ヘッダー（共通コンポーネント）

#### PC版

**1段目:**
| 要素 | 詳細 |
|------|------|
| ロゴ | クリックでトップページ（C-01）へ |
| 検索バー | キーワード入力 → 商品一覧ページ（C-02）へ遷移 |
| ご利用ガイド | C-17へ |
| FAQ | C-16へ |
| 入稿について | C-17（ご利用ガイド内の入稿セクション）へ |
| ログイン / マイページ | 未ログイン時:「ログイン」を表示（C-08へ）。ログイン済み:「マイページ」を表示（C-10へ）。表示が切り替わる |
| カート | C-04へ。カート内商品数をバッジ表示 |

**2段目（カテゴリナビ）:**
| 要素 | 詳細 |
|------|------|
| カテゴリ名 | 横並びで表示 |
| ホバー時 | ドロップダウンでカテゴリ内の商品を表示（画像＋商品名＋説明の3列グリッド） |
| カテゴリ名クリック | C-02（該当カテゴリの商品一覧）へ遷移 |
| ドロップダウン内の商品クリック | 該当商品のC-03（商品詳細ページ）へ遷移 |

#### SP版

**1段目:**
| 要素 | 詳細 |
|------|------|
| ロゴ | クリックでトップページ（C-01）へ |
| ログイン / マイページ | アイコン＋テキスト。PC版と同じ切替仕様 |
| カート | アイコン＋テキスト＋バッジ。C-04へ |
| メニュー | アイコン＋テキスト。タップでメニューを展開（カテゴリ一覧・ご利用ガイド・FAQ等） |

**2段目:**
| 要素 | 詳細 |
|------|------|
| 検索バー＋検索ボタン | キーワード入力 → C-02へ遷移 |

> **タップ領域:** 全ナビアイコンのタップ領域は44x44px以上を確保

### 3.2 商品画像ギャラリー（PC: 右カラム・sticky / SP: 最上部）

| 要素 | 詳細 | 設定元 | PC | SP |
|------|------|--------|----|----|
| メイン画像 | オプション選択に連動して切り替わる | 商品別 | 表示 | 全幅表示。スワイプ・ピンチズーム対応 |
| サムネイル一覧 | 正面・裏面・使用例・印刷範囲 等。タップで切替 | 商品別 | 表示 | 非表示 |
| ドットインジケーター | 画像枚数分のドット。スワイプ位置を示す | — | 非表示 | 表示 |

### 3.3 商品情報（PC: 左カラム / SP: 画像下）

#### 3.3.1 商品ヘッダー

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| カテゴリ名 | 商品が属するカテゴリ。クリックでC-02へ | 商品別 |
| 商品名 | h1。商品の正式名称 | 商品別 |
| 商品説明 | 商品の概要テキスト。SP版は2行で折りたたみ、「もっと見る」で展開 | 商品別 |

#### 3.3.2 価格表示（PC: 右カラム内 / SP: 商品情報内＋下部CTAバー）

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 合計金額（税込） | 単価系オプションの合計 × 数量。固定費は含まない | 自動算出 |
| 1個あたり単価 | 右寄せで表示 | 自動算出 |
| 納期目安 | 左:「納期目安」ラベル、右: 太字で日数表示（例: 約7営業日） | 商品別 |
| 内訳展開 | 「内訳を見る」クリックで展開/折りたたみ | — |

**内訳の構成:**

| 区分 | 内容 | 設定元 |
|------|------|--------|
| 単価内訳 | 本体単価（数量範囲表示）＋各オプション追加単価。合計で単価合計を表示 | 商品別・自動算出 |
| 固定費 | 色校正、入稿サポート等。選択なしの場合は「--」表示 | 商品別 |
| 送料 | この商品の送料を表示。合計金額がN円以上の場合は「無料」と表示。N円未満の場合は「500円（N円以上のご注文で送料無料）」と表示。オプション・数量変更に連動してリアルタイム更新 | 商品別＋共通（無料閾値） |

#### 3.3.3 オプション選択エリア

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 共通仕様 | 全オプションは必須。各オプションにデフォルト値を設定し、ページ表示時点で価格が即座に表示される | 商品別（デフォルト値を管理者が設定） |
| ヘルプアイコン | 各オプション項目名の横に「?」アイコンを表示（任意）。クリックで吹き出しツールチップ表示。もう一度クリックで閉じる。ヘルプテキスト未設定の場合はアイコン非表示 | 商品別（項目ごとにヘルプテキストを任意設定） |

**オプションのパターン一覧:**

| パターン | 画像 | 価格タイプ | ヘルプ | 例 |
|----------|------|-----------|--------|-----|
| A | なし（テキストのみ） | なし | あり | サイズ |
| B | あり | なし | あり | 素材・仕上げ |
| C | あり | 単価系（+N円） | あり | 付属パーツ |
| D | なし（テキストのみ） | 単価系（+N円） | なし | 包装 |
| E | なし（テキストのみ） | 固定費（+N円） | あり | 色校正（あり/なし） |
| F | なし（テキストのみ） | 固定費（+N円） | あり | 入稿サポート（あり/なし） |

> **単価系 vs 固定費:** 単価系は数量に比例（単価に加算）。固定費は数量に関わらず一律。

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| テキストオプション | ボタン形式で選択。選択中は枠線＋背景で強調。追加価格ありの場合は「(+10円)」を併記 | 商品別 |
| 画像付きオプション | PC: 4列グリッド、SP: 3列グリッド。画像＋名前＋追加価格（任意） | 商品別 |
| バッジ | 「人気」等のラベル表示（任意） | 商品別 |
| 数量入力 | 数値入力フィールド。最小ロット以上なら任意の数量で注文可能 | — |
| 最小ロット表示 | 数量入力の下に「最小ロット: 100個」等を表示 | 商品別 |

#### 3.3.4 アクションボタン・サブリンク（PC: 右カラム / SP: 下部CTAバー＋セクション内）

**カートに追加・サンプル制作ボタン:**

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| カートに追加ボタン | プライマリボタン。クリック → カート追加完了モーダル表示 | 固定テキスト |
| サンプル制作ボタン | セカンダリボタン。サンプル制作1式（10個）としてカートに追加 | 固定テキスト |

> SP版ではこの2ボタンは下部CTAバー（sticky）に配置。価格も併記。

**サブリンク（3列グリッド・アイコン＋テキスト）:**

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 見積書をダウンロード | 現在のオプション・数量で見積書PDFを生成・ダウンロード | — |
| 入稿テンプレートをダウンロード | 選択中のオプションに対応するテンプレートファイルをダウンロード | 商品別 |
| この商品について問い合わせる | C-14へ遷移（商品情報が自動セット） | — |

> アイコンはプレースホルダー（実装時に適切なアイコンに差し替え）

### 3.4 価格テーブル・商品詳細・入稿仕様（PC: タブ / SP: アコーディオン）

PC版はタブ切替、SP版はアコーディオン（折りたたみ）で表示。

#### 価格テーブル

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| テーブル | 数量範囲別の単価一覧（税込）。「100〜199」「200〜299」のように範囲表記。最小ロット以上なら任意の数量で注文可能。SP版は横スクロール対応 | 商品別 |
| 注記 | テーブル下に注意事項を表示 | 商品別 |

#### 商品詳細

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 詳細テーブル | 素材、サイズ展開、仕上げ、最小ロット、納期目安 等 | 商品別 |
| 表示形式 | ラベル（左）＋値（右）の2列テーブル | — |
| 注記 | テーブル下に注意事項を表示 | 商品別 |

#### 入稿仕様

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 入稿仕様本文 | リッチテキスト（WYSIWYG）で自由に記述。印刷方法、対応形式、カラーモード、解像度、注意事項等を商品に応じて記載 | 商品別 |
| テンプレートDL | 入稿テンプレートのダウンロードリンク（選択中のオプションに対応） | 商品別 |

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| オプション選択 | 商品画像が連動して切り替わる。価格・納期・内訳がリアルタイム更新 |
| 数量入力 | 価格・納期がリアルタイム更新。任意の数量を入力可能（最小ロット以上） |
| ヘルプアイコンクリック | アイコン近くに吹き出しツールチップで説明を表示。もう一度クリックで閉じる |
| 内訳を見る | 価格の内訳（単価内訳＋固定費）を展開/折りたたみ |
| カートに追加 | ログイン不要。カートに追加完了モーダルを表示。未ログイン時はブラウザにローカル保存 |
| サンプル制作をカートに追加 | 上記と同様。カートにサンプル種別（1式10個）で追加。ログイン不要 |
| 見積書DL | 現在のオプション・数量で見積書PDFを生成・ダウンロード |
| テンプレートDL | 選択中のオプションに対応するテンプレートファイルをダウンロード |
| 問い合わせ | C-14へ遷移（商品情報が自動セット） |
| タブ切替（PC） | 価格テーブル / 商品詳細 / 入稿仕様 を切替表示 |
| アコーディオン開閉（SP） | 価格テーブル / 商品詳細 / 入稿仕様 を展開/折りたたみ |
| もっと見る（SP） | 商品説明の全文を展開/折りたたみ |
| FAQアコーディオン | クリックで開閉 |
| 関連商品クリック | 該当商品のC-03へ遷移 |
| カテゴリホバー（PC） | ドロップダウンで商品一覧表示 |
| 画像スワイプ（SP） | 画像を左右スワイプで切替。ドットインジケーターが連動 |
| ピンチズーム（SP） | 画像を拡大表示 |

**カート追加完了モーダル（PC: 中央 / SP: ボトムシート）:**

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| タイトル | 「商品をカートに入れました」 | 固定テキスト |
| カート情報 | 合計点数、合計金額（税込） | 自動算出 |
| 「買い物を続ける」ボタン | モーダルを閉じてC-03に留まる | 固定テキスト |
| 「カートを見る」ボタン | C-04（カートページ）へ遷移 | 固定テキスト |
| 閉じるボタン（×） | モーダルを閉じる | — |

## 6. データ要件

この画面の表示に必要なデータ。設定元が「商品別」のものは管理者が商品ごとに設定、「共通」は全商品で同じ内容が表示される。

| データ | 内容 | 設定元 |
|--------|------|--------|
| 商品名 | 商品の正式名称 | 商品別 |
| 商品説明文 | 商品の概要テキスト | 商品別 |
| カテゴリ | 商品が属するカテゴリ | 商品別 |
| 商品画像 | 複数枚。オプションとの紐づけ情報 | 商品別 |
| オプション定義 | オプショングループごとの選択肢。各選択肢に画像（任意）・追加価格・バッジ・デフォルト値・ヘルプテキスト（任意）。価格タイプ（単価系/固定費）を含む | 商品別 |
| 価格テーブル | 数量範囲別の単価一覧 | 商品別 |
| 価格テーブル注記 | 価格テーブル下の注意事項テキスト | 商品別 |
| 納期テーブル | 数量別の納期目安 | 商品別 |
| 最小ロット | 通常注文の最小数量 | 商品別 |
| サンプル制作価格 | サンプル1式（10個）の価格 | 商品別 |
| 商品詳細情報 | 素材、サイズ展開、仕上げ等の仕様テーブル | 商品別 |
| 商品詳細注記 | 商品詳細下の注意事項テキスト | 商品別 |
| 入稿仕様 | リッチテキスト。印刷方法、対応形式、カラーモード、解像度等 | 商品別 |
| 入稿テンプレート | オプション別のテンプレートファイル | 商品別 |
| 関連商品 | 関連商品のリスト（商品ID、画像、名前、最低単価） | 商品別 |
| 個別商品FAQ | この商品に紐づくFAQ（質問・回答のリスト） | 商品別（ADM-001） |
| グローバル商品FAQ | 全商品共通のFAQ（質問・回答のリスト） | 共通（ADM-008） |
