## 2. 画面構成

### PC版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | C-03と同じ |
| パンくずリスト | トップ > マイページ > 会員情報 |

| 会社情報 | 表示モード / 編集モード |
| 住所（配送先） | 表示モード / 編集モード |
| アカウント | 表示のみ |
| パンくずリスト（下部） | フッター直上 |
| フッター（共通） | C-03と同じ |

### SP版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | C-03 SP版と同じ |
| 「マイページに戻る」バー | サイドナビの代わり |
| 会社情報 | 表示モード / 編集モード |
| 住所（配送先） | 表示モード / 編集モード |
| アカウント | 表示のみ |
| フッター（共通） | C-03 SP版と同じ |

**SP版でのPC版との主な差分：**
- サイドナビ → 「マイページに戻る」バー
- パンくず非表示
- フォームの姓名が縦並び

## 3. セクション詳細

### 3.1 表示モード（デフォルト）

初期表示は登録済みの情報をテーブル形式で表示。編集はできない。

#### 会社情報

| 項目 | 表示 |
|------|------|
| 会社名 | テキスト表示 |
| 担当者名 | 姓＋名をまとめて表示 |
| 電話番号 | テキスト表示 |
| 「編集」リンク | セクションタイトル横に配置。クリックで編集モードに切替 |

#### 住所（配送先）

| 項目 | 表示 |
|------|------|
| 郵便番号 | テキスト表示 |
| 都道府県 | テキスト表示 |
| 市区町村・番地 | テキスト表示 |
| 建物名 | テキスト表示 |
| 「編集」リンク | セクションタイトル横に配置。クリックで編集モードに切替 |

#### アカウント

| 項目 | 表示 |
|------|------|
| メールアドレス | 表示のみ。編集不可 |
| ログイン方法 | 「Googleアカウントでログイン中」等を表示 |

### 3.2 編集モード（「編集」クリック後）

「編集」をクリックすると全セクションがフォーム入力に切り替わる。

#### フォーム項目

| 項目 | 必須/任意 | 入力形式 | 備考 |
|------|----------|----------|------|
| 会社名 | 必須 | テキスト | — |
| 担当者名（姓） | 必須 | テキスト | 姓名を分割 |
| 担当者名（名） | 必須 | テキスト | — |
| 電話番号 | 必須 | テキスト（tel） | — |
| 郵便番号 | 必須 | テキスト | 「住所検索」ボタンで自動入力 |
| 都道府県 | 必須 | プルダウン | 郵便番号から自動入力 |
| 市区町村・番地 | 必須 | テキスト | 郵便番号から一部自動入力 |
| 建物名・部屋番号 | 任意 | テキスト | — |

#### アクションボタン

| 要素 | 詳細 |
|------|------|
| 「変更を保存」ボタン | クリックで保存。成功時「保存しました」メッセージを表示し、表示モードに戻る |
| 「キャンセル」リンク | 変更を破棄して表示モードに戻る |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 「編集」クリック | 表示モード → 編集モードに切替 |
| 「変更を保存」クリック | バリデーション実行。成功 → 保存して表示モードに戻り「保存しました」表示。失敗 → エラー表示 |
| 「キャンセル」クリック | 変更を破棄して表示モードに戻る |
| 「住所検索」クリック | 郵便番号から都道府県・市区町村を自動入力 |

---

## 5. バリデーション・エラー

| 条件 | 表示 |
|------|------|
| 必須項目が未入力 | 該当項目の枠線が赤に。下に「この項目は必須です」 |
| 電話番号の形式不正 | 「正しい電話番号を入力してください」 |
| 郵便番号の形式不正 | 「正しい郵便番号を入力してください」 |

---

## 6. 補足

- 会社情報未登録の状態で入稿・注文確定しようとした場合、この画面にリダイレクトされる
- C-06（注文確認ページ）の「配送先を変更」からもこの画面に遷移する
- 表示/編集モードの切替により、保存ボタンを押さずにページを離脱してもデータが変更されない
