## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-11 |
| 画面名 | 会員情報編集ページ |
| URL | `/mypage/profile` |
| 認証 | 必要 |
| 対応デバイス | PC・スマートフォン（レスポンシブ） |

---

## 2. セクション詳細

### 2.1 会社情報フォーム

| 項目 | 必須/任意 | 入力形式 |
|------|----------|----------|
| 会社名 | 必須 | テキスト |
| 担当者名（姓） | 必須 | テキスト |
| 担当者名（名） | 必須 | テキスト |
| 電話番号 | 必須 | テキスト（tel） |

### 2.2 住所（配送先）フォーム

| 項目 | 必須/任意 | 入力形式 | 備考 |
|------|----------|----------|------|
| 郵便番号 | 必須 | テキスト | 「住所検索」ボタンで自動入力 |
| 都道府県 | 必須 | プルダウン | 郵便番号から自動入力 |
| 市区町村・番地 | 必須 | テキスト | 郵便番号から一部自動入力 |
| 建物名・部屋番号 | 任意 | テキスト | — |

### 2.3 アカウント情報（表示のみ）

| 項目 | 詳細 |
|------|------|
| メールアドレス | 表示のみ。編集不可 |
| ログイン方法 | 「Googleアカウントでログイン中」等を表示 |

### 2.4 アクション

| 要素 | 詳細 |
|------|------|
| 「変更を保存」ボタン | クリックで保存。成功時「保存しました」メッセージ表示 |

---

## 3. 補足

- 会社情報未登録の状態で入稿・注文確定しようとした場合、この画面にリダイレクトされる
- C-06（注文確認ページ）の「配送先を変更」からもこの画面に遷移する
