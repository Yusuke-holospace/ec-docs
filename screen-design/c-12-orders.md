---

## 2. セクション詳細

### 2.1 フィルター

| フィルター | 内容 |
|------------|------|
| すべて | 全注文を表示（デフォルト） |
| 通常注文 | 通常注文のみ |
| サンプル制作 | サンプル制作のみ |
| 進行中 | 完了以外のステータス |
| 完了 | 完了ステータスのみ |

> SP版ではフィルターボタンが横スクロール

### 2.2 注文カード

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 注文番号 | ORD-XXXXXXXX-XXXX | サーバー |
| 注文日 | YYYY/MM/DD | サーバー |
| ステータスバッジ | 入稿確認中（青）/ 決済待ち（オレンジ）/ 完了（緑）等 | サーバー |
| 商品概要 | 商品名＋数量の簡潔な一覧 | サーバー |
| 注文種別 | 「通常注文」「サンプル制作」「通常＋サンプル」 | サーバー |
| 合計金額 | 税込 | サーバー |
| クリック | C-13（注文詳細）へ遷移 | — |

---

## 3. 補足

- 注文が0件の場合は「注文履歴がありません」＋「商品を探す」ボタンを表示
