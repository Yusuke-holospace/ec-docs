---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-09 |
| 画面名 | 会員登録ページ |
| URL | `/register` |
| 認証 | 不要（ソーシャルログイン or マジックリンク認証後に遷移） |

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー | ミニマル（ロゴのみ・中央寄せ） |
| 登録フォーム | タイトル・説明・会社情報入力フォーム |
| アクションボタン | 「登録を完了する」＋「あとで入力する」 |
| フッター（共通） | C-03と同じ |

---

## 3. セクション詳細

### 3.1 タイトル・説明

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| タイトル | 「会社情報の登録」 | 固定テキスト |
| 説明文 | 「ご注文に必要な会社情報を入力してください。初回のみの登録です。」 | 固定テキスト |

### 3.2 フォーム項目

| 項目 | 必須/任意 | 入力形式 | 備考 |
|------|----------|----------|------|
| 会社名 | 必須 | テキスト | — |
| 担当者名（姓） | 必須 | テキスト | 姓名を分割 |
| 担当者名（名） | 必須 | テキスト | — |
| 電話番号 | 必須 | テキスト（tel） | — |
| 郵便番号 | 必須 | テキスト | 「住所検索」ボタンで都道府県・市区町村を自動入力 |
| 都道府県 | 必須 | プルダウン | 郵便番号から自動入力 |
| 市区町村・番地 | 必須 | テキスト | 郵便番号から一部自動入力 |
| 建物名・部屋番号 | 任意 | テキスト | — |

### 3.3 アクション

| 要素 | 詳細 |
|------|------|
| 「登録を完了する」ボタン | 全必須項目が入力済みの場合のみ活性。クリックで登録完了しC-10（マイページ）or 元の画面へ遷移 |
| 「あとで入力する」リンク | スキップしてC-10 or 元の画面へ遷移。注記「入稿・注文確定には会社情報の登録が必要です」 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 「住所検索」クリック | 郵便番号から都道府県・市区町村を自動入力 |
| 「登録を完了する」クリック | 会社情報を保存。登録完了メール（NTF-001）を送信。遷移先は元の画面（入稿時はC-05、通常はC-10） |
| 「あとで入力する」クリック | 会社情報未登録のまま遷移。入稿・注文確定時にC-11（会員情報編集ページ）に遷移して入力を求める |

---

## 5. バリデーション・エラー

| 条件 | 表示 |
|------|------|
| 必須項目が未入力 | 該当項目の下に赤字で「この項目は必須です」 |
| 電話番号の形式不正 | 「正しい電話番号を入力してください」 |
| 郵便番号の形式不正 | 「正しい郵便番号を入力してください」 |
| 郵便番号で住所が見つからない | 「該当する住所が見つかりません」 |

---

## 6. 表示タイミング

| タイミング | 動作 |
|------------|------|
| 初回ログイン時 | C-08でログイン後、自動でC-09に遷移 |
| 会社情報未登録で入稿・注文確定しようとした時 | C-11（会員情報編集ページ）に遷移（C-09ではない） |
