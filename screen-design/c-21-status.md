# 画面設計書：C-21 共有ステータスページ

> ワイヤーフレーム PC: `wireframes/c-21-status.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/c-21-status.html)
> ワイヤーフレーム SP: `wireframes/c-21-status-sp.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/c-21-status-sp.html)

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-21 |
| 画面名 | 共有ステータスページ |
| URL | `/orders/status/[token]` |
| 認証 | **不要**（トークン付きURLでアクセス） |
| 対応デバイス | PC・スマートフォン（レスポンシブ） |

---

## 2. 画面の目的

- EC注文・代理注文を問わず、注文ステータスをログイン不要で確認できるページ
- 顧客が社内の他の担当者にURLを共有して進捗確認に使う
- ステータス変更通知メール（NTF-011）からのリンク先

---

## 3. C-13（注文詳細ページ）との違い

| 観点 | C-13 注文詳細 | C-21 共有ステータス |
|------|-------------|-------------------|
| 認証 | ログイン必要 | トークンURLのみ（不要） |
| ヘッダー | EC共通ヘッダー（2段構成） | ミニマルヘッダー（ロゴ + ECサイトへリンク） |
| フッター | EC共通フッター（3カラム） | ミニマルフッター（コピーライトのみ） |
| サイドナビ | マイページナビ | なし |
| パンくず | あり | なし |
| 再入稿ボタン | あり（差し戻し時） | **なし**（閲覧専用） |
| リピート注文ボタン | あり（完了時） | **なし** |
| 共有URLコピー | あり | **なし**（このページ自体が共有先） |
| 入稿状態 | 表示あり | **なし**（ステータスのみ） |

## 4. セクション詳細

### 4.1 ヘッダー（ミニマル）

| 要素 | 詳細 |
|------|------|
| ロゴ | ECサイトトップページへのリンク |
| 「ECサイトへ」リンク | ECサイトトップページ（C-01）へ遷移 |

### 4.2 ステータス

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| ステータスバッジ | 現在のステータスを色付きバッジで表示 | サーバー |
| 注文日 | YYYY/MM/DD | サーバー |
| 注文番号 | ORD-XXXXXXXX-XXXX | サーバー |

### 4.3 ステータスタイムライン

C-13と同じ形式。完了済みステップは緑、現在のステップは黒、未到達はグレー。各ステップに日時を表示（完了済みのみ）。

| ステップ | 表示 |
|----------|------|
| 注文受付 | 入稿受付 or サンプル受付 |
| 入稿確認中 | 通常注文のみ |
| 承認済み | 通常注文のみ |
| 決済待ち | — |
| 入金確認 | — |
| 製造中 | — |
| 発送済み | — |
| 完了 | — |

### 4.4 注文商品一覧

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 商品画像 | サムネイル | 商品別 |
| 種別バッジ | 通常注文 / サンプル制作 | サーバー |
| 商品名 | — | 商品別 |
| オプション | サイズ / 仕上げ / 数量 | 注文別 |
| 金額 | 単価 × 数量 | 注文別 |

> C-13と異なり、入稿状態（入稿済み/未入稿）は表示しない

### 4.5 お支払い

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 支払い方法 | クレジットカード / 銀行振込 / 請求書払い | サーバー |
| 商品合計 | 全商品の小計合計 | サーバー |
| 送料 | 無料 or 金額 | サーバー |
| 合計（税込） | 商品合計＋送料 | サーバー |

### 4.6 配送先

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 住所 | 会社名・担当者名・住所・電話番号 | サーバー |

### 4.7 仕上がりイメージ（条件表示）

管理者が仕上がりイメージをアップロード済みの場合のみ表示。注文商品一覧の直後に配置。

| 要素 | 詳細 |
|------|------|
| サムネイル一覧 | 3列グリッドで画像を表示 |
| 説明テキスト | 「販促用素材としてご利用いただけます（背景透過PNG含む）」 |
| 「一括ダウンロード（ZIP）」ボタン | 全画像をZIPにまとめてダウンロード |

### 4.8 お問い合わせ

| 要素 | 詳細 |
|------|------|
| テキスト | 「ご注文に関するご質問は、下記までお問い合わせください。」 |
| メールアドレス | support@holospace.jp |
| 補足 | 「注文番号をお伝えいただくとスムーズです。」 |

> C-13の「担当者からのご連絡」セクションに代わり、こちらではお問い合わせ先を表示

### 4.9 フッター（ミニマル）

| 要素 | 詳細 |
|------|------|

## 6. セキュリティ・アクセス制御

| 項目 | 詳細 |
|------|------|
| トークン | 注文ごとにUUID v4で自動生成。推測不可能な文字列 |
| 有効期限 | なし（注文が存在する限りアクセス可能。要検討） |
| 無効なトークン | C-20（404エラーページ）にリダイレクト |
| 表示制限 | 閲覧のみ。編集・操作は一切不可 |

---

## 7. 補足

- このページ自体に「共有URLをコピー」ボタンは不要（ブラウザのURL欄がそのまま共有URL）
- ステータス変更通知メール（NTF-011）内のリンクからアクセスされることを想定
- C-07（注文完了ページ）およびC-13（注文詳細ページ）の「共有URLをコピー」ボタンから、このページのURLがコピーされる
- 代理注文（EC外注文）の場合、管理画面A-07からもこのURLを確認・コピーできる
