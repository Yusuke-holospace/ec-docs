## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | C-06 |
| 画面名 | 注文確認ページ |
| URL | `/cart/confirm` |
| 認証 | 必要 |
| 対応デバイス | PC・スマートフォン（レスポンシブ） |

---

## 2. 画面構成

### PC版

| セクション | 内容 |
|------------|------|
| ヘッダー（共通） | C-03と同じ |
| パンくずリスト | トップ > カート > 注文確認 |
| ステップインジケーター | カート → 入稿 → **注文確認** → 完了 |
| 注文商品一覧 | 全商品の情報・入稿状態・金額 |
| 配送先 | 登録住所＋変更リンク |
| データ確認後のご連絡先 | メールアドレス・電話番号（登録情報から自動入力、編集可能） |
| お支払い方法 | クレカ・銀行振込・請求書払いから選択 |
| お支払い金額 | 商品合計・送料・納期目安・合計・注意事項 |
| ライセンス同意 | 同意文＋チェックボックス |
| 注文後の流れ（アコーディオン） | デフォルト閉じ。展開で4ステップ表示。薄い青背景 |
| アクションボタン | 「カートに戻る」「注文を確定する」＋決済注記 |
| パンくずリスト（下部） | フッター直上 |
| フッター（共通） | C-03と同じ |

### SP版

PC版と同じセクション構成。差分：
- パンくず非表示
- ステップインジケーターはコンパクト
- ボタン縦並び（プライマリが上）

## 3. セクション詳細

### 3.1 ステップインジケーター

| ステップ | 状態 |
|----------|------|
| 1. カート | 完了（緑） |
| 2. データ入稿 | 完了（緑） |
| 3. 注文確認 | 現在（黒・アクティブ） |
| 4. 完了 | 未到達（グレー） |

### 3.2 注文商品一覧

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 商品画像 | サムネイル | 商品別 |
| 注文種別バッジ | 「通常注文」/「サンプル制作」 | カートデータ |
| 商品名 | 商品の正式名称 | 商品別 |
| オプション・数量 | 選択済みオプション＋数量を表示 | カートデータ |
| 入稿状態 | 「入稿済み」（ファイル名は非表示） | サーバー |
| 金額 | 小計（単価×数量＋固定費） | 自動算出 |
| 内訳 | 単価×数量、固定費がある場合はその内容 | 自動算出 |

### 3.3 配送先

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 住所表示 | 会社名・お名前・郵便番号・住所・電話番号 | 会員情報 |
| 変更リンク | C-11（会員情報編集ページ）へ遷移。変更後は手動でC-06に戻る | — |

### 3.4 データ確認後のご連絡先

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 説明文 | 「入稿データの確認結果やお支払いのご案内をお送りします。」 | 固定テキスト |
| メールアドレス | テキスト入力。会員登録情報からデフォルト入力。編集可能 | 会員情報 |
| 電話番号 | テキスト入力。会員登録情報からデフォルト入力。編集可能 | 会員情報 |
| 補足 | 「会員登録情報から自動入力されています。変更した場合、この注文のみに適用されます。」 | 固定テキスト |

### 3.5 お支払い方法

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| クレジットカード | Visa, Mastercard, JCB, AMEX。Stripe経由 | 共通 |
| 銀行振込 | 振込先情報をメールで送信 | 共通 |
| 請求書払い | 月末締め翌月払い。請求書払いサービス経由 | 共通 |
| 選択UI | ラジオボタン。選択中は枠線で強調 | — |
| 注意書き | 「入稿データの確認後に、ご登録のメールアドレスにお支払いのご案内をお送りします。」「お支払い完了後に製造開始となります。」 | 固定テキスト |

### 3.6 お支払い金額

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 商品合計（N点） | 全商品の小計合計 | 自動算出 |
| 送料 | 各商品の送料合算。合計がN円以上なら「無料」 | 商品別＋共通（無料閾値） |
| 納期目安 | カート内全商品の中で最も長い納期を表示。「約X営業日」形式。「入金確認完了後から起算」の補足付き | 商品別（商品・数量に応じた納期設定） |
| 合計（税込） | 商品合計＋送料 | 自動算出 |
| 注意事項 | 注文確定後の流れ（入稿確認→決済案内）。「この時点では決済は発生しません」を明記 | 固定テキスト |

### 3.7 ライセンス同意

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 同意文 | ライセンスに関する同意内容。スクロール可能なボックスで表示 | 固定テキスト |
| チェックボックス | 「上記ライセンスに関する内容に同意します」。チェックしないと「注文を確定する」ボタンが非活性 | — |

### 3.8 注文後の流れ（アコーディオン）

| 要素 | 詳細 |
|------|------|
| 配置 | 注文確定ボタンの直上 |
| デフォルト状態 | 閉じた状態。「注文後の流れを確認 ▼」 |
| 背景色 | 薄い青（#f0f7ff、ボーダー: #d0e3f7） |
| 展開時の内容 | 4ステップの番号付きリスト: ①入稿データ確認（数営業日）→ ②決済案内メール送信 → ③支払い確認後製造開始 → ④製造完了後発送 |

### 3.9 アクションボタン

| 要素 | 詳細 |
|------|------|
| カートに戻る | C-04へ遷移 |
| 注文を確定する | ライセンス同意チェック済みの場合のみ活性。クリックでC-07（注文完了ページ）へ遷移 |
| 決済注記 | ボタン近くに「この時点では決済は発生しません。入稿データ確認後に決済のご案内をお送りします。」 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| ライセンス同意チェック | 「注文を確定する」ボタンが活性化 |
| 「注文を確定する」クリック | 注文を確定しC-07（注文完了ページ）へ遷移。メール通知（NTF-002, NTF-008）が送信される |
| 「カートに戻る」クリック | C-04へ遷移 |
| 「配送先を変更」クリック | C-11（会員情報編集ページ）へ遷移 |
| 連絡先（メール・電話）編集 | 入力値を変更可能。変更内容はこの注文のみに適用（会員情報は更新されない） |
| 「注文後の流れを確認」クリック | アコーディオンを展開/折りたたみ |

---

## 5. データ要件

| データ | 内容 | 設定元 |
|--------|------|--------|
| カート内商品リスト | 全商品の情報・オプション・数量・入稿状態・金額 | カートデータ＋商品別 |
| 配送先住所 | 会員の登録住所 | 会員情報 |
| 連絡先（メール・電話） | 会員の登録情報をデフォルト表示。注文ごとに変更可能 | 会員情報 |
| 送料 | 各商品の送料＋無料閾値 | 商品別＋共通 |
| ライセンス同意文 | 同意テキスト | 固定テキスト |
