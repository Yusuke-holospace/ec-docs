# 画面設計書：A-01 管理者ログインページ

> ワイヤーフレーム: `wireframes/a-01-login.html` — [プレビュー](https://yusuke-holospace.github.io/ec-docs/wireframes/a-01-login.html)
> PC専用（管理画面はレスポンシブ対象外）

---

## 1. 画面概要

| 項目 | 内容 |
|------|------|
| 画面ID | A-01 |
| 画面名 | 管理者ログインページ |
| URL | `/admin/login` |
| 認証 | 不要（ログイン画面自体） |

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー | ミニマル（ロゴのみ・中央寄せ。管理画面用ダーク背景 #1a1a2e） |
| ログインボックス | タイトル・説明・メール入力・パスワード入力・ログインボタン |
| フッター | コピーライトのみ |

---

## 3. セクション詳細

### 3.1 ログインフォーム

| 要素 | 詳細 |
|------|------|
| タイトル | 「管理者ログイン」 |
| 説明文 | 「管理画面にアクセスするにはログインしてください。」 |
| メールアドレス | テキスト入力（email） |
| パスワード | パスワード入力 |
| ログインボタン | 全幅。管理画面テーマカラー（#1a1a2e） |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| ログインボタンクリック | メール+パスワードで認証。成功 → A-02（ダッシュボード）へ遷移。失敗 → エラーメッセージ表示 |

---

## 5. バリデーション・エラー

| 条件 | 表示 |
|------|------|
| メールアドレス未入力 | 「メールアドレスを入力してください」 |
| パスワード未入力 | 「パスワードを入力してください」 |
| 認証失敗 | 「メールアドレスまたはパスワードが正しくありません。」 |

---

## 6. 補足

- 管理画面のテーマカラーは #1a1a2e（ダークネイビー）。購入者画面（#333）と区別
- パスワードリセット機能は初期スコープ外（管理者間で直接対応）
