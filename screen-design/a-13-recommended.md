---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル | 「おすすめ商品管理」 |
| 説明テキスト | 「C-01 トップページの「おすすめ商品」セクションに表示される商品を管理します。」 |
| 商品追加セクション | 商品検索＋検索結果から追加 |
| おすすめ商品リスト | ドラッグ並び替え可能なリスト |

---

## 3. セクション詳細

### 3.1 商品追加セクション

| 要素 | 詳細 |
|------|------|
| 商品名検索 | テキスト入力。登録済み商品から部分一致検索 |
| カテゴリフィルタ | プルダウン。カテゴリで絞り込み |
| 検索結果 | サムネイル＋商品名＋カテゴリ＋「追加」ボタン |

### 3.2 おすすめ商品リスト

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| ドラッグハンドル | 並び順変更用のグリップアイコン | — |
| 並び順番号 | 現在の表示順 | 管理者 |
| 商品サムネイル | 商品画像 | サーバー |
| 商品名 | 商品名称 | サーバー |
| カテゴリ | 所属カテゴリ名 | サーバー |
| 削除ボタン | おすすめリストから削除 | — |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 商品検索 | テキスト入力・カテゴリ選択で登録済み商品を絞り込み |
| 「追加」クリック | 検索結果の商品をおすすめリストの末尾に追加 |
| ドラッグ&ドロップ | 表示順を変更。変更は自動保存。C-01の表示順に即時反映 |
| 「削除」クリック | おすすめリストから商品を削除（商品自体は削除されない） |
