## 3. セクション詳細

### 3.1 ツールバー

| 要素 | 詳細 |
|------|------|
| 検索 | 注文番号・顧客名で部分一致検索 |
| ステータスフィルタ | 全10ステータス（入稿受付〜完了）。**注文内のいずれかの商品が該当ステータスを持つ注文**を抽出（例: 「差し戻し」を選ぶと差し戻し中の商品を含む注文がヒット） |
| 種別フィルタ | すべて / 通常注文 / サンプル制作 |
| チャネルフィルタ | すべて / EC / 電話 / メール / 営業 |
| 日付範囲フィルタ | 開始日〜終了日の日付入力。注文日で絞り込み |
| 表示件数 | 右端に「全XX件」 |

### 3.2 注文テーブル

| 列 | 内容 | 設定元 |
|----|------|--------|
| 注文番号 | ORD-XXXXXXXX-XXXX | サーバー |
| 顧客名 | 会社名 or お名前 | サーバー |
| 商品 | 商品名＋数量の簡潔表示。商品が複数の場合は「商品A 他N点」 | サーバー |
| 金額 | 合計（税込） | サーバー |
| ステータス | 注文内の**各商品ステータスをバッジ並列で表示**（同一ステータスはグループ化して件数を併記）。商品が1点の場合はバッジ1つ。色定義はC-12と同じ | サーバー |
| 種別 | 通常 / サンプル / 通常+サンプル | サーバー |
| チャネル | EC / 電話 / メール / 営業（色分けバッジ） | サーバー |
| 担当 | 担当スタッフ名。未割り当ては「—」 | サーバー |
| 注文日 | MM/DD | サーバー |

**ステータス内訳の表示ルール:**

- 同一ステータスの商品をグループ化し、`[ステータス名 ×件数]` 形式で表示
- 商品が1点の場合は件数を省略し `[ステータス名]` のみ
- 並び順: ステータス進行順（入稿受付/サンプル受付 → 入稿確認中 → 差し戻し → 承認済み → 決済待ち → 入金確認 → 製造中 → 発送済み → 完了）

例:
- 商品1点（入稿確認中） → `[入稿確認中]`
- 商品3点（入稿確認中=2、差し戻し=1） → `[入稿確認中 ×2] [差し戻し ×1]`

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 注文行クリック | A-07（注文詳細）へ遷移 |
| 「代理注文作成」クリック | 代理注文作成フォームを表示（A-07内 or モーダル） |
| 「CSV出力」クリック | 現在のフィルター条件で注文データをCSVダウンロード |
| 検索・フィルタ | 入力・選択で即時絞り込み |
| A-02の通知カードから遷移 | 該当ステータスのフィルタが適用された状態で表示 |
