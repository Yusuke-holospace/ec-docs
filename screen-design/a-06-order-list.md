## 1. 画面概要

| 項目 | 内容 |
|------|------|

---

## 2. 画面構成

| セクション | 内容 |
|------------|------|
| ヘッダー・サイドナビ | 共通レイアウト（A-02参照） |
| ページタイトル＋アクション | CSV出力＋代理注文作成ボタン |
| ツールバー | 検索・ステータスフィルタ・種別フィルタ・チャネルフィルタ |
| 注文テーブル | 一覧表示 |
| ページネーション | ページ番号ナビ |

---

## 3. セクション詳細

### 3.1 ツールバー

| 要素 | 詳細 |
|------|------|
| 検索 | 注文番号・顧客名で部分一致検索 |
| ステータスフィルタ | 全10ステータス（入稿受付〜完了） |
| 種別フィルタ | すべて / 通常注文 / サンプル制作 |
| チャネルフィルタ | すべて / EC / 電話 / メール / 営業 |
| 表示件数 | 右端に「全XX件」 |

### 3.2 注文テーブル

| 列 | 内容 | 設定元 |
|----|------|--------|
| 注文番号 | ORD-XXXXXXXX-XXXX | サーバー |
| 顧客名 | 会社名 or お名前 | サーバー |
| 商品 | 商品名＋数量の簡潔表示 | サーバー |
| 金額 | 合計（税込） | サーバー |
| ステータス | 色付きバッジ（C-12と同じ色定義） | サーバー |
| 種別 | 通常 / サンプル / 通常+サンプル | サーバー |
| チャネル | EC / 電話 / メール / 営業（色分けバッジ） | サーバー |
| 担当 | 担当スタッフ名。未割り当ては「—」 | サーバー |
| 注文日 | MM/DD | サーバー |

### 3.3 ページネーション

| 要素 | 詳細 |
|------|------|
| 表示件数 | 1ページあたり20件 |
| ナビ | ページ番号＋「次へ」 |

---

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 注文行クリック | A-07（注文詳細）へ遷移 |
| 「代理注文作成」クリック | 代理注文作成フォームを表示（A-07内 or モーダル） |
| 「CSV出力」クリック | 現在のフィルター条件で注文データをCSVダウンロード |
| 検索・フィルタ | 入力・選択で即時絞り込み |
| A-02の通知カードから遷移 | 該当ステータスのフィルタが適用された状態で表示 |
