## 3. ダッシュボード固有のセクション

### 3.1 通知カード

対応が必要な項目の件数を4枚のカードで表示。クリックでA-06（該当フィルター適用済み）に遷移。

| カード | カウント対象 | 設定元 |
|--------|------------|--------|
| 新規注文（未確認） | ステータスが「入稿受付」または「サンプル受付」の注文数。管理者がまだ確認作業に入っていない注文 | サーバー |
| 入稿確認待ち | ステータスが「入稿確認中」の注文数。管理者が入稿データを確認中で、承認/差し戻しの判断待ち | サーバー |
| 決済待ち | ステータスが「決済待ち」の注文数。決済案内送信済みで、購入者の支払いを待っている状態 | サーバー |
| 差し戻し（再入稿待ち） | ステータスが「差し戻し」の注文数。1つ以上の商品が差し戻され、購入者の再入稿を待っている状態 | サーバー |

### 3.2 最近の注文

| 要素 | 詳細 | 設定元 |
|------|------|--------|
| 表示件数 | 最新5件 | サーバー |
| 列 | 注文番号・顧客名・商品・金額・ステータス・チャネル・注文日時 | サーバー |
| 行クリック | A-07（注文詳細）へ遷移 | — |
| 「すべて見る」リンク | A-06（注文一覧）へ遷移 | — |

## 4. インタラクション

| アクション | 動作 |
|------------|------|
| 通知カードクリック | A-06に遷移（該当ステータスでフィルター適用） |

| 「すべて見る」クリック | A-06（注文一覧）へ遷移 |
| サイドナビクリック | 各管理画面へ遷移 |
| ログアウトクリック | セッション終了、A-01へ遷移 |
